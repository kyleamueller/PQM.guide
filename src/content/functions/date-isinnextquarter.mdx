## Remarks

`Date.IsInNextQuarter` returns `true` if the input date falls within the calendar quarter immediately following the current quarter (as determined by `DateTime.LocalNow()`). For example, if today is in Q1, this function returns `true` for any date in Q2. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter next quarter's pipeline

Keep only opportunities expected to close in the next quarter.

```powerquery
Table.SelectRows(
    Pipeline,
    each Date.IsInNextQuarter([ExpectedClose])
)
```

### Example 2: Flag next-quarter opportunities

Add a boolean column identifying next-quarter deals.

```powerquery

    Pipeline,
    "NextQuarter", each Date.IsInNextQuarter([ExpectedClose]), type logical
)
```
