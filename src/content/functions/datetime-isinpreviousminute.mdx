## Remarks

`DateTime.IsInPreviousMinute` returns `true` if the input `datetime` or `datetimezone` value falls within the clock minute immediately preceding the current minute (as determined by `DateTime.LocalNow()`). For example, if it is currently 10:32, this function returns `true` for any datetime between 10:31:00 and 10:31:59.9999999. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter events from the previous minute

Keep only rows where EventTime falls in the previous clock minute.

```powerquery
Table.SelectRows(
    Events,
    each DateTime.IsInPreviousMinute([EventTime])
)
```

### Example 2: Flag previous-minute events

Add a boolean column for events in the previous minute.

```powerquery
Table.AddColumn(
    Events,
    "IsPreviousMinute", each DateTime.IsInPreviousMinute([EventTime]), type logical
)
```
