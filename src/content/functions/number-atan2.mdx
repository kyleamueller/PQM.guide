## Remarks

`Number.Atan2(y, x)` computes the angle θ such that the point (x, y) lies on the ray from the origin at angle θ from the positive x-axis. The result is in the range (-π, π].

Note that the parameter order is **(y, x)** — y first, then x. This matches the mathematical convention and matches Excel's `ATAN2` argument order (Excel also takes y first as the numerator).

Unlike `Number.Atan(y/x)`, which loses quadrant information when both y and x are negative, `Number.Atan2` correctly distinguishes all four quadrants and handles the case where x = 0.

## Examples

### Example 1: Angle to the point (1, 1) — 45 degrees

```powerquery
Number.Atan2(1, 1) * 180 / Number.PI
```

### Example 3: Bearing angle from origin to a map coordinate

Compute the compass bearing in degrees to a point at (x=3, y=4):

```powerquery
let
    X = 3,
    Y = 4,
    AngleDegrees = Number.Atan2(Y, X) * 180 / Number.PI
in
    AngleDegrees
```
