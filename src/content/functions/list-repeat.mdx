## Remarks

`List.Repeat` concatenates the input list with itself `count` times, producing a longer list. This is useful for creating repeated patterns, padding lists, or generating cyclic sequences.

- `count` must be a non-negative integer.
- If `count` is `0`, an empty list is returned.
- If `count` is `1`, a copy of the original list is returned.
- The result has `List.Count(list) * count` elements.

## Examples

### Example 1: Repeat a list 3 times

```powerquery
List.Repeat({1, 2, 3}, 3)
```

### Example 2: Create a repeating pattern of labels

```powerquery
List.Repeat({"Q1", "Q2", "Q3", "Q4"}, 3)
```

<!--output

### Example 3: Pad a list with a repeated filler value

```powerquery
let
    FillerList = List.Repeat({null}, 5)
in
    FillerList
```
