## Remarks

`DateTime.IsInNextMinute` returns `true` if the input `datetime` or `datetimezone` value falls within the clock minute immediately following the current minute (as determined by `DateTime.LocalNow()`). For example, if it is currently 10:32, this function returns `true` for any datetime between 10:33:00 and 10:33:59.9999999. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter events in the next minute

Keep only rows where EventTime falls in the next clock minute.

```powerquery
Table.SelectRows(
    Events,
    each DateTime.IsInNextMinute([EventTime])
)
```

<!--output

### Example 2: Flag next-minute events

Add a boolean column for events in the next minute.

```powerquery
Table.AddColumn(
    Events,
    "IsNextMinute", each DateTime.IsInNextMinute([EventTime]), type logical
)
```
