## Remarks

`Date.AddWeeks` adds or subtracts a whole number of weeks (7-day periods) from a `date`, `datetime`, or `datetimezone` value. `Date.AddWeeks(d, n)` is equivalent to `Date.AddDays(d, n * 7)`. Passing a negative `numberOfWeeks` subtracts weeks. The return type matches the input type.

## Examples

### Example 1: Shift OrderDate forward by 2 weeks

Add a 2-week lead time to each order date.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "ShipByDate", each Date.AddWeeks([OrderDate], 2), type date
)
```

### Example 2: Subtract one week from a date

Find the date one week before a reference date.

```powerquery
#table(
    type table [Today = date, LastWeek = date],
    {{#date(2024, 3, 15), Date.AddWeeks(#date(2024, 3, 15), -1)}}
)
```
