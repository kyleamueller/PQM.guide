## Remarks

`Text.PadEnd` adds `character` characters to the right (end) of `text` until the total length is at least `count`. If `text` is already `count` characters or longer, it is returned unchanged.

If `character` is omitted or `null`, a space (`" "`) is used. The `character` argument must be exactly one character long; providing a multi-character string raises an error.

If `text` is `null`, the function returns `null`.

Right-padding is commonly used for fixed-width text file output, aligning columns in reports, or filling codes to a standard length.

## Examples

### Example 1: Pad a label to ten characters with spaces

```powerquery
Text.PadEnd("Hello", 10)
```
