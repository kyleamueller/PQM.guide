## Remarks

`Text.ReplaceRange` removes `count` characters from `text` starting at the zero-based `offset` position, then inserts `newText` at that same position. The replacement text does not need to be the same length as the removed portion.

If `text` is `null`, the function returns `null`. The length of `newText` may differ from `count`, allowing the result to be shorter or longer than the original.

This is equivalent to: `Text.Start(text, offset) & newText & Text.End(text, Text.Length(text) - offset - count)`

Use `Text.Replace` when you want to replace by value (all occurrences of a substring). Use `Text.ReplaceRange` when you want to replace by position.

## Examples

### Example 1: Replace a word at a known position

```powerquery
Text.ReplaceRange("Hello World", 6, 5, "There")
```

### Example 2: Replace a separator

```powerquery

```

### Example 3: Replace a prefix code

```powerquery

```

### Example 4: Replace with a longer string (expansion)

```powerquery
Text.ReplaceRange("AB12", 2, 2, "9999")
```
