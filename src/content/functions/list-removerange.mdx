## Remarks

`List.RemoveRange` removes a contiguous block of items from a list, starting at the given zero-based `index`. The `count` parameter specifies how many items to remove; if omitted, a single item is removed.

- An `index` of `0` starts removal from the first item.
- If `count` extends beyond the end of the list, all items from `index` onward are removed.
- The original list is not modified; a new list is returned.
- Use `List.InsertRange` or `List.ReplaceRange` for inserting or replacing ranges.

## Examples

### Example 1: Remove 2 items starting at index 1

```powerquery
List.RemoveRange({1, 2, 3, 4, 5}, 1, 2)
```

### Example 2: Remove a single item (default count)

```powerquery
List.RemoveRange({"A", "B", "C", "D"}, 2)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{A, B, D}"}]}

### Example 3: Remove from the middle of a list

```powerquery
let
    Months = {"Jan", "Feb", "Mar", "Apr", "May", "Jun"},
    WithoutQ2 = List.RemoveRange(Months, 3, 3)
in
    WithoutQ2
```
