## Remarks

`List.Durations` generates a sequence of `duration` values. Duration values represent a span of time specified as days, hours, minutes, and seconds using the `#duration(days, hours, minutes, seconds)` literal.

This function is useful for generating time intervals for scheduling analysis, elapsed time buckets, or SLA calculations.

- `count` must be a non-negative integer; 0 returns an empty list.
- The `start` and `step` values use the `#duration` literal syntax.
- Duration arithmetic is exact — there are no calendar irregularities (unlike months).

## Examples

### Example 1: Generate hourly durations for 5 hours

```powerquery
List.Durations(#duration(0, 0, 0, 0), 5, #duration(0, 1, 0, 0))
```
