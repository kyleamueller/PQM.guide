## Remarks

`List.AllTrue` returns `true` if every item in the list is logically true. It returns `false` as soon as any item is `false`. Null values in the list are treated as false.

This function is commonly used to validate that a list of boolean results — such as the output of `List.Transform` applying a condition — all pass.

- An empty list returns `true` (vacuous truth).
- Null values cause the function to return `false`.
- Non-logical values may cause an error; use `List.Transform` to coerce values first if needed.

## Examples

### Example 1: All values are true

```powerquery
List.AllTrue({true, true, true})
```

### Example 3: Check all quantities are positive

```powerquery
let
    Quantities = {10, 25, 5, 8},
    AllPositive = List.AllTrue(List.Transform(Quantities, each _ > 0))
in
    AllPositive
```
