## Remarks

`Text.Start` extracts a substring from the beginning of a text value. It returns the first `count` characters of `text`.

If `text` is `null`, the function returns `null`. If `count` is greater than the length of `text`, the entire text is returned without error. If `count` is 0, an empty text string `""` is returned.

This function is commonly used to extract prefixes, abbreviations, or fixed-width fields from the left side of a string.

## Examples

### Example 1: Get the first five characters of a string

```powerquery
Text.Start("Hello World", 5)
```
