## Remarks

`List.IsDistinct` checks whether every item in the list is unique. It returns `true` if all items are distinct, and `false` if any value appears more than once.

This is a convenient shorthand for checking `List.Count(list) = List.Count(List.Distinct(list))`, but may be more efficient.

- An empty list returns `true` (vacuous truth — no duplicates exist).
- Null values are treated as equal to each other, so two nulls make the list non-distinct.
- Pass `Comparer.OrdinalIgnoreCase` for case-insensitive text comparison.

## Examples

### Example 1: All values are distinct

```powerquery
List.IsDistinct({1, 2, 3, 4, 5})
```

### Example 3: Case-insensitive check


### Example 4: Validate a key column has no duplicates

```powerquery

    Keys = {"ID-001", "ID-002", "ID-003", "ID-004"},
    IsUnique = List.IsDistinct(Keys)

    IsUnique
```
