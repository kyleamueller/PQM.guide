## Remarks

`Text.Repeat` concatenates `text` with itself `count` times. If `count` is 0, an empty string is returned. If `text` is `null`, the function returns `null`.

Common uses include generating separators, padding patterns, or building test data strings. When combined with `Text.Length`, it can be used to create fixed-length fill strings.

## Examples

### Example 1: Repeat a string three times

```powerquery
Text.Repeat("abc", 3)
```
