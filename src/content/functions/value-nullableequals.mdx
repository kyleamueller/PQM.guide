## Remarks

`Value.NullableEquals` implements SQL-style three-valued equality: when either operand is `null`, the result is `null` rather than `true` or `false`. This matches the behavior of the M `=` operator for nullable values.

This contrasts with `Value.Equals(null, null)` which returns `true` regardless of null inputs.

Use `Value.NullableEquals` when you need standard SQL-style null propagation, such as when building JOIN-like logic or filtering comparisons that must propagate unknown (null) results.
