## Remarks

`Number.BitwiseAnd` performs a bitwise AND between two integers. Each bit position in the result is set to `1` only when the corresponding bit is `1` in both operands. This operation is commonly used for:

- **Masking**: extracting specific bits from a value by ANDing with a bitmask
- **Flag testing**: checking whether specific flag bits are set
- **Permission systems**: determining whether a user has a specific permission

Both inputs are treated as integers. Floating-point values are truncated to their integer portion before the operation.

## Examples

### Example 1: Basic bitwise AND

12 in binary is `1100`, 10 is `1010`. AND gives `1000` = 8.

```powerquery
Number.BitwiseAnd(12, 10)
```

### Example 2: Test if a value has a specific bit set (bit 2 = value 4)


### Example 3: Extract low nibble (lower 4 bits) using mask 0xF = 15

