## Remarks

`Number.Asin` is the inverse of `Number.Sin`. Given a value in [-1, 1], it returns the angle in radians whose sine equals that value. The result is in the range [-π/2, π/2].

Input values outside [-1, 1] return `NaN`. To convert the result from radians to degrees, multiply by `180 / Number.PI`.

## Examples

### Example 1: Arcsine of 1 (90 degrees)

```powerquery
Number.Asin(1)
```
