## Remarks

`Date.IsInNextNDays` returns `true` if the input date falls between tomorrow and N days from today (inclusive). Today itself is not included; use `Date.IsInCurrentDay` if you also need to match today. The function is re-evaluated on each query refresh.

For example, with `days = 7`, it matches any date from tomorrow through 7 days from now (a 7-day window starting tomorrow).

## Examples

### Example 1: Find tasks due in the next 14 days

Keep only rows where DueDate falls within the next 14 days.

```powerquery
Table.SelectRows(
    Tasks,
    each Date.IsInNextNDays([DueDate], 14)
)
```

### Example 2: Flag rows expiring in the next 30 days

Add a boolean column for items expiring within 30 days.

```powerquery
Table.AddColumn(
    Licenses,
    "ExpiringSoon", each Date.IsInNextNDays([ExpiryDate], 30), type logical
)
```
