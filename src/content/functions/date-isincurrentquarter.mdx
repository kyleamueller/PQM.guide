## Remarks

`Date.IsInCurrentQuarter` returns `true` if the input date falls within the same calendar quarter as today (`DateTime.LocalNow()`). Quarters are defined as: Q1 (Jan–Mar), Q2 (Apr–Jun), Q3 (Jul–Sep), Q4 (Oct–Dec). The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter this quarter's sales

Keep only rows where OrderDate falls in the current quarter.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInCurrentQuarter([OrderDate])
)
```

### Example 2: Flag current-quarter rows

Add a boolean column to identify rows from the current quarter.

```powerquery
Table.AddColumn(
    Sales,
    "IsCurrentQuarter", each Date.IsInCurrentQuarter([OrderDate]), type logical
)
```
