## Remarks

`Int32.From` converts a value to a signed 32-bit integer, the most common integer size in databases and programming. It covers the range -2,147,483,648 to 2,147,483,647. Values outside this range raise an error. Fractional numbers are rounded (default: round-toward-even).

Text values are parsed numerically. `true` converts to `1`, `false` to `0`, `null` to `null`.

This maps to `int` in SQL Server, `INTEGER` in SQLite, and similar types across many systems. Use `Int64.From` for larger integers.
