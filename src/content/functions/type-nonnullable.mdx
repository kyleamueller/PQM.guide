## Remarks

`Type.NonNullable` returns the non-nullable form of a type. If the input type is `nullable X`, the result is `X`. If the input is already non-nullable, it is returned unchanged.

This is useful when you need to compare or work with type values in a normalized form, stripping any nullable wrapper before inspection.

The inverse operation — making a type nullable — is performed with `type nullable X` syntax rather than a dedicated function.

## Examples
