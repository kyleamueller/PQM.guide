## Remarks

`Number.Acos` is the inverse of `Number.Cos`. Given a value in [-1, 1], it returns the angle in radians whose cosine equals that value. The result is in the range [0, π].

Input values outside [-1, 1] return `NaN`. To convert radians to degrees, multiply by `180 / Number.PI`.

## Examples

### Example 1: Arccosine of 1 (0 degrees)

```powerquery
Number.Acos(1)
```
