## Remarks

`Text.PositionOf` searches for `substring` within `text` and returns the zero-based character position of the match. Returns `-1` if the substring is not found.

The optional `occurrence` parameter accepts values from the `Occurrence` enum:
- `Occurrence.First` (default, value `0`) — returns the position of the first match
- `Occurrence.Last` (value `1`) — returns the position of the last match
- `Occurrence.All` (value `2`) — returns a list of all match positions

The optional `comparer` parameter controls how character comparison is performed. By default, an ordinal (byte-by-byte) comparison is used, which is case-sensitive. Pass `Comparer.OrdinalIgnoreCase` for case-insensitive searches.

Use `Text.Contains` when you only need to know whether a substring exists, not its position.

## Examples

### Example 1: Find the position of the first occurrence

```powerquery
Text.PositionOf("Hello World Hello", "Hello")
```
