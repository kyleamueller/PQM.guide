## Remarks

`Date.IsInCurrentMonth` returns `true` if the input date falls within the same year and month as today's date (`DateTime.LocalNow()`). It matches the entire calendar month from the first to the last day, inclusive. This function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter this month's orders

Keep only rows from the Sales table where OrderDate is in the current month.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInCurrentMonth([OrderDate])
)
```

### Example 2: Flag current-month rows

Add a boolean column identifying rows in the current month.

```powerquery
Table.AddColumn(
    Sales,
    "IsCurrentMonth", each Date.IsInCurrentMonth([OrderDate]), type logical
)
```
