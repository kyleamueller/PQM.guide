## Remarks

`Text.BeforeDelimiter` returns all text that precedes the specified `delimiter` occurrence. By default it uses the first occurrence (index 0).

The `index` parameter can be:
- A non-negative integer: the zero-based occurrence to use (0 = first, 1 = second, etc.)
- A list `{n, RelativePosition.FromEnd}`: count occurrences from the end of the string

If the delimiter is not found, the entire text is returned. If `text` is `null`, the function returns `null`.

This function is commonly used in the Power Query UI when splitting or extracting columns by delimiter.

## Examples

### Example 1: Text before the first hyphen

```powerquery
Text.BeforeDelimiter("US-WA-Seattle", "-")
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"US"}]}
-->

### Example 2: Text before the second hyphen

```powerquery
Text.BeforeDelimiter("US-WA-Seattle", "-", 1)
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"US-WA"}]}
-->

### Example 3: Text before the last hyphen (from end)

```powerquery
Text.BeforeDelimiter("US-WA-Seattle", "-", {0, RelativePosition.FromEnd})
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"US-WA"}]}
-->

### Example 4: Extract username from email address

```powerquery
Text.BeforeDelimiter("user@example.com", "@")
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"user"}]}
-->
