## Remarks

`Date.IsInYearToDate` returns `true` if the input date falls between January 1 of the current year and today's date (inclusive), as determined by `DateTime.LocalNow()`. This differs from `Date.IsInCurrentYear`, which includes the entire year including future dates. Use this function for YTD metrics where you only want to compare the portion of the year that has elapsed. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter year-to-date sales

Keep only rows where OrderDate falls within the current year up to today.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInYearToDate([OrderDate])
)
```

### Example 2: Compare YTD vs. current year rows

Show the difference between YTD and full current-year flags.

```powerquery
Table.AddColumn(
    Table.AddColumn(
        Sales,
        "IsCurrentYear", each Date.IsInCurrentYear([OrderDate]), type logical
    ),
    "IsYTD", each Date.IsInYearToDate([OrderDate]), type logical
)
```
