## Remarks

`Duration.TotalMinutes` returns the entire duration as a decimal number of minutes. For example, 2 hours and 30 minutes returns `150.0`. This differs from `Duration.Minutes`, which returns only the minutes component of the duration. Use `Duration.TotalMinutes` when you need elapsed time as a single numeric value for SLA analysis, billing calculations, or reporting. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Calculate response time in minutes from log data

Compute the total minutes elapsed between log start and end times.

Input: `OrderLog`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.SelectRows(OrderLog, each [DurationMinutes] <> null),
        {"LogID", "DurationMinutes"}
    ),
    "TotalMinutes",
    each Duration.TotalMinutes(#duration(0, 0, [DurationMinutes], 0)),
    type number
)
```

### Example 2: Total minutes for a mixed duration

Return the total minutes for a duration of 2 hours and 30 minutes.

```powerquery
#table(
    type table [Duration = duration, TotalMinutes = number],
    {{#duration(0, 2, 30, 0), Duration.TotalMinutes(#duration(0, 2, 30, 0))}}
)
```
