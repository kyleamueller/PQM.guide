## Remarks

`Date.StartOfMonth` returns the first day of the month for the given `date`, `datetime`, or `datetimezone` value. The return type matches the input type. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Find the start of each order's month

Get the first day of the month for each OrderDate in the Sales table.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "MonthStart", each Date.StartOfMonth([OrderDate]), type date
)
```

<!--output
{"columns":[{"name":"OrderID","type":"number"},{"name":"OrderDate","type":"date"},{"name":"MonthStart","type":"date"}],"rows":[{"OrderID":1,"OrderDate":"2024-01-15","MonthStart":"2024-01-01"},{"OrderID":2,"OrderDate":"2024-01-18","MonthStart":"2024-01-01"},{"OrderID":3,"OrderDate":"2024-02-01","MonthStart":"2024-02-01"},{"OrderID":4,"OrderDate":"2024-02-10","MonthStart":"2024-02-01"},{"OrderID":5,"OrderDate":"2024-03-05","MonthStart":"2024-03-01"}]}
-->

### Example 2: Start of month from a scalar value

Return the first day of the month for a single date.

```powerquery
#table(
    type table [Date = date, StartOfMonth = date],
    {{#date(2024, 4, 15), Date.StartOfMonth(#date(2024, 4, 15))}}
)
```

<!--output
{"columns":[{"name":"Date","type":"date"},{"name":"StartOfMonth","type":"date"}],"rows":[{"Date":"2024-04-15","StartOfMonth":"2024-04-01"}]}
-->
