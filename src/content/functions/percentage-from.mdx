## Remarks

`Percentage.From` converts a value to the percentage data type in Power Query M. The underlying numeric value is stored as a decimal (e.g., `0.25` represents 25%). The percentage type is primarily a display annotation — it signals to the host application (such as Power BI Desktop) that the value should be formatted as a percentage.

When converting from text, the value is interpreted as a raw decimal unless the text includes a `%` symbol. For example, `"25%"` converts to `0.25` in some environments.

Text values are parsed using the specified culture. `true` converts to `1`, `false` to `0`, `null` to `null`.

## Examples

### Example 1: Convert a decimal to percentage type

```powerquery
Percentage.From(0.25)
```

<!--output

### Example 2: Convert text

```powerquery
Percentage.From("0.75")
```

<!--output

### Example 3: Add a percentage column to a table

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Sales, 3), {"OrderID", "Discount"}),
    "DiscountPct", each Percentage.From([Discount]), Percentage.Type
)
```
