## Remarks

`List.Difference` performs a set-difference operation: it returns all items in `list1` that are not present in `list2`. Duplicate items in `list1` are preserved if they are not in `list2`.

This function is useful for finding new or removed items when comparing two lists — for example, finding products that exist in last month's data but not this month's.

- Comparison is case-sensitive for text values by default.
- Pass `Comparer.OrdinalIgnoreCase` as `equationCriteria` for case-insensitive text comparison.
- Order of items in the result follows the order in `list1`.
- Unlike `List.RemoveItems`, this function works on a full set-difference basis and uses value equality rather than reference equality.

## Examples

### Example 1: Find items in list1 not in list2

```powerquery
List.Difference({1, 2, 3, 4, 5}, {2, 4})
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{1, 3, 5}"}]}

### Example 2: Find discontinued products

```powerquery
let
    LastMonth = {"Widget A", "Widget B", "Widget C", "Widget D"},
    ThisMonth = {"Widget A", "Widget C"},
    Discontinued = List.Difference(LastMonth, ThisMonth)
in
    Discontinued
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{Widget B, Widget D}"}]}
-->

### Example 3: Case-insensitive difference

```powerquery
List.Difference({"Apple", "Banana", "Cherry"}, {"apple", "cherry"}, Comparer.OrdinalIgnoreCase)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{Banana}"}]}
-->
