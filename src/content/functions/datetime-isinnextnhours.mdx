## Remarks

`DateTime.IsInNextNHours` returns `true` if the input `datetime` or `datetimezone` value falls within the next N complete clock hours after the current hour. The current hour itself is not included; use `DateTime.IsInCurrentHour` to also match the current hour. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter events in the next 4 hours

Keep only rows where EventTime falls within the next 4 clock hours.

```powerquery
Table.SelectRows(
    Events,
    each DateTime.IsInNextNHours([EventTime], 4)
)
```

### Example 2: Flag events in the next 8 hours

Add a boolean column identifying events within the next 8 hours.

```powerquery
Table.AddColumn(
    Events,
    "InNext8Hours", each DateTime.IsInNextNHours([EventTime], 8), type logical
)
```
