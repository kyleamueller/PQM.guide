## Remarks

`Text.End` extracts a substring from the end of a text value. It returns the last `count` characters of `text`.

If `text` is `null`, the function returns `null`. If `count` is greater than the length of `text`, the entire text is returned. If `count` is 0, an empty text string `""` is returned.

This function is commonly used to extract suffixes, file extensions, or the trailing portion of a formatted string such as a phone number or product code.

## Examples

### Example 1: Get the last four characters of a string

```powerquery
Text.End("Hello World", 5)
```
