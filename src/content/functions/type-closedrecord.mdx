## Remarks

`Type.ClosedRecord` converts a record type into a closed record type. A closed record type requires that values have exactly the fields declared in the type — no additional fields are permitted.

If the type is already closed, it is returned unchanged. This complements `Type.OpenRecord`.

Closed record types are useful for strict schema validation and for signaling that a record is fully described by its declared fields.

## Examples

### Example 1: Close the base record type

```powerquery
let
    Closed = Type.ClosedRecord(type record)
in
    Type.IsOpenRecord(Closed)
```
