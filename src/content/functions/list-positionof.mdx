## Remarks

`List.PositionOf` searches a list for a specified value and returns its zero-based index. By default, it finds the first occurrence.

- Returns `-1` if the value is not found in the list.
- Use `Occurrence.First` (default) to find the first match.
- Use `Occurrence.Last` to find the last match.
- Use `Occurrence.All` to return a list of **all** positions where the value appears.
- Comparison is exact and case-sensitive for text by default. Pass `Comparer.OrdinalIgnoreCase` as `equationCriteria` for case-insensitive search.

## Examples

### Example 1: Find the position of a value

```powerquery
List.PositionOf({"A", "B", "C", "D"}, "C")
```
