## Remarks

`Character.FromNumber` converts a Unicode code point (a non-negative integer) into the corresponding single-character text value. This is the inverse of `Character.ToNumber`.

Common code point ranges include:
- 0–31: Control characters (tab = 9, newline = 10, carriage return = 13)
- 32: Space
- 48–57: Digits `0`–`9`
- 65–90: Uppercase letters `A`–`Z`
- 97–122: Lowercase letters `a`–`z`

This function is useful for generating special characters programmatically, building character sequences with `List.Transform`, or inserting control characters such as line feeds or tabs into text values.

## Examples

### Example 1: Convert code point 65 to a character

```powerquery
Character.FromNumber(65)
```

### Example 2: Generate the tab character


### Example 3: Build the alphabet as a list

```powerquery
List.Transform({65..90}, Character.FromNumber)
```

### Example 4: Round-trip with Character.ToNumber

```powerquery
Character.FromNumber(Character.ToNumber("M"))
```
