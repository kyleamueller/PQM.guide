## Remarks

`Number.BitwiseXor` performs a bitwise exclusive OR. A bit in the result is `1` when the corresponding bits in the two operands are different — one is `0` and the other is `1`. When they are the same, the result bit is `0`.

XOR has a useful property: `a XOR b XOR b = a`. This means XOR-ing a value twice with the same mask returns the original value — making it useful for simple toggling of flags or for checking whether two values differ at the bit level.

## Examples

### Example 1: Basic bitwise XOR

12 (`1100`) XOR 10 (`1010`) = `0110` = 6.

```powerquery
Number.BitwiseXor(12, 10)
```

### Example 2: Toggle a bit (bit 1 = value 2)

```powerquery
Number.BitwiseXor(5, 2)
```

### Example 3: XOR self-inverse — XOR twice returns the original

```powerquery
let
    Original = 42,
    Mask = 15,
    Encoded = Number.BitwiseXor(Original, Mask),
    Decoded = Number.BitwiseXor(Encoded, Mask)
in
    Decoded
```
