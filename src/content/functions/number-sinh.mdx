## Remarks

`Number.Sinh` returns the hyperbolic sine, defined as:

sinh(x) = (e^x − e^(−x)) / 2

Unlike the circular sine, hyperbolic sine is unbounded — it increases without limit as x grows. `Number.Sinh(0)` returns `0`. The function is an odd function: `sinh(−x) = −sinh(x)`.

Hyperbolic functions appear in physics (catenary curves, special relativity), engineering, and certain statistical distributions.
