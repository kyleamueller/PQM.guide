## Remarks

`Number.Tanh` returns the hyperbolic tangent, defined as:

tanh(x) = sinh(x) / cosh(x) = (e^x − e^(−x)) / (e^x + e^(−x))

The result is always in the open interval (-1, 1), approaching -1 as x → -∞ and 1 as x → +∞. `Number.Tanh(0)` returns `0`.

Hyperbolic tangent is widely used as an activation function in neural networks and for normalizing values to (-1, 1) without a hard cutoff.
