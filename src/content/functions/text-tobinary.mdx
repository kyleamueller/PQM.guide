## Remarks

`Text.ToBinary` encodes a text value into a binary representation using the specified character encoding. This is the inverse of `Text.FromBinary`.

The `encoding` parameter accepts values from the `TextEncoding` enum:
- `TextEncoding.Utf8` (65001) — default
- `TextEncoding.Ascii` (20127)
- `TextEncoding.Unicode` / `TextEncoding.Utf16` (1200)
- `TextEncoding.Utf16BE` (1201)
- `TextEncoding.Utf32` (12000)
- `TextEncoding.Windows` (1252)

Common uses include preparing text for binary storage, generating file content, computing checksums or hashes, or creating payloads for web APIs that expect binary-encoded data.

## Examples

### Example 1: Encode text as UTF-8

```powerquery
Text.ToBinary("Hello World")
```

### Example 2: Encode text with explicit UTF-8 encoding

```powerquery
Text.ToBinary("Power Query", TextEncoding.Utf8)
```

### Example 3: Round-trip encode and decode

```powerquery
let
    original = "Hello",
    encoded = Text.ToBinary(original, TextEncoding.Utf8),
    decoded = Text.FromBinary(encoded, TextEncoding.Utf8)
in
    decoded
```
