## Remarks

`Record.RenameFields` returns a new record with specified fields renamed. The rename list is a list of two-element lists where the first element is the existing field name and the second is the desired new name.

The optional `missingField` parameter controls behavior when an old name is not found:
- `MissingField.Error` (default, value 0) — raises an error
- `MissingField.Ignore` (value 1) — silently skips missing fields
- `MissingField.UseNull` (value 2) — adds the new field with a null value

Field values and order are preserved; only the names change.

## Examples

### Example 1: Rename a single field

```powerquery
Record.RenameFields([Name = "Alice", Age = 30], {{"Name", "FullName"}})
```

<!--output
{"columns":[{"name":"FullName","type":"text"},{"name":"Age","type":"number"}],"rows":[{"FullName":"Alice","Age":30}]}
-->

### Example 2: Rename multiple fields at once

```powerquery
Record.RenameFields(
    [Nm = "Bob", Ag = 25, Ct = "Seattle"],
    {{"Nm", "Name"}, {"Ag", "Age"}, {"Ct", "City"}}
)
```

<!--output
{"columns":[{"name":"Name","type":"text"},{"name":"Age","type":"number"},{"name":"City","type":"text"}],"rows":[{"Name":"Bob","Age":25,"City":"Seattle"}]}
-->

### Example 3: Ignore a missing field gracefully

```powerquery
Record.RenameFields(
    [Name = "Alice", Age = 30],
    {{"Name", "FullName"}, {"Email", "EmailAddress"}},
    MissingField.Ignore
)
```

<!--output
{"columns":[{"name":"FullName","type":"text"},{"name":"Age","type":"number"}],"rows":[{"FullName":"Alice","Age":30}]}
-->
