## Remarks

`List.MinN` returns the bottom `N` values from a list. The result is a list (not a scalar), sorted in ascending order. It is the multi-value counterpart of `List.Min`.

- When passed a number `N`, it returns the `N` smallest values.
- When passed a condition function, it returns values sorted ascending while the condition is true (applied to the sorted list from the bottom).
- Null values are ignored in the comparison.
- If `N` exceeds the list length, the entire list is returned sorted ascending.
- Duplicates are preserved.

## Examples

### Example 1: Return the 3 smallest numbers

```powerquery
List.MinN({15, 3, 42, 8, 27, 1}, 3)
```

### Example 2: Return values while they are below a threshold

```powerquery
List.MinN({10, 20, 30, 40, 50}, each _ <= 30)
```

### Example 3: Find the 3 lowest-cost items

```powerquery
let
    Costs = {1200, 850, 3400, 450, 2100, 975},
    BottomThree = List.MinN(Costs, 3)
in
    BottomThree
```
