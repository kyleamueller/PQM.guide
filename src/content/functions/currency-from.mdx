## Remarks

`Currency.From` converts a value to the currency data type, which is a fixed-decimal number with exactly 4 decimal places. The underlying representation is a scaled 64-bit integer (value × 10,000), giving exact arithmetic for monetary amounts up to ±922,337,203,685,477.5807.

The currency type prevents floating-point rounding errors that plague financial calculations when using double-precision arithmetic. It maps to the `Currency` type in Power BI Desktop and `Fixed Decimal Number` in the UI.

Text values are parsed using the specified culture. `true` converts to `1.0000`, `false` to `0.0000`, `null` to `null`.

## Examples

### Example 1: Convert a number to currency

```powerquery
Currency.From(19.99)
```

### Example 3: Apply currency type to a revenue column

Input: `Sales`

```powerquery
Table.TransformColumnTypes(
    Table.SelectColumns(Table.FirstN(Sales, 3), {"OrderID", "Revenue"}),
    {{"Revenue", Currency.Type}}
)
```
