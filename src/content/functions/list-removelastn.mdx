## Remarks

`List.RemoveLastN` removes the last `N` items from a list and returns the remaining items. It accepts either a count or a condition function.

- **Number**: Removes the last `N` items. If `N` is larger than the list, an empty list is returned.
- **Condition function**: Removes items from the **end** of the list while the condition returns `true`, scanning backward.
- The original list is not modified; a new list is returned.

## Examples

### Example 1: Remove the last 2 items

```powerquery
List.RemoveLastN({1, 2, 3, 4, 5}, 2)
```

<!--output

### Example 2: Remove trailing items while value exceeds 30

```powerquery
List.RemoveLastN({10, 20, 30, 40, 50}, each _ > 30)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{10, 20, 30}"}]}

### Example 3: Remove footer rows from a list

```powerquery
let
    Rows = {"Data1", "Data2", "Data3", "---", "Footer", "Total"},
    DataOnly = List.RemoveLastN(Rows, 3)
in
    DataOnly
```
