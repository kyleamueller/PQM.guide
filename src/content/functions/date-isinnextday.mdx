## Remarks

`Date.IsInNextDay` returns `true` if the input date falls on the calendar day immediately following today's date (`DateTime.LocalNow() + 1 day`). It does not include today itself. The function is re-evaluated on each query refresh, making it useful for highlighting upcoming deadlines or scheduled items.

## Examples

### Example 1: Filter tomorrow's scheduled tasks

Keep only rows where DueDate is tomorrow.

```powerquery
Table.SelectRows(
    Tasks,
    each Date.IsInNextDay([DueDate])
)
```

### Example 2: Flag tomorrow's rows

Add a boolean column identifying rows due tomorrow.

```powerquery

    Tasks,
    "DueTomorrow", each Date.IsInNextDay([DueDate]), type logical
)
```
