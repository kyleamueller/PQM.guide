## Remarks

`Table.IsEmpty` returns `true` if the table has no rows. This is more readable and efficient than checking `Table.RowCount(table) = 0`. It is commonly used in conditional logic â€” for example, checking whether a filter returned any results before performing further operations.

## Examples

### Example 1: Check if filtered table is empty

Filter Sales for a non-existent region and check whether the result is empty.

Input: `Sales`

```powerquery
let
    Filtered = Table.SelectRows(Sales, each [Region] = "South"),
    IsEmpty = Table.IsEmpty(Filtered)
in
    #table({"IsEmpty"}, {{IsEmpty}})
```

### Example 2: Check a non-empty table

Check whether the full Sales table is empty.

Input: `Sales`

```powerquery
let
    IsEmpty = Table.IsEmpty(Sales)
in
    #table({"IsEmpty"}, {{IsEmpty}})
```
