## Remarks

`Single.From` converts a value to a single-precision (32-bit) floating-point number. Single-precision provides approximately 7 significant decimal digits of precision, less than the standard double-precision type. Values that exceed the single-precision range (~±3.4 × 10^38) will overflow to infinity.

This type is less commonly used in Power Query than `Double.From`, but may be needed when interfacing with systems or data sources that explicitly use 32-bit floats.

Text values are parsed using the specified culture. `true` converts to `1.0`, `false` to `0.0`, `null` to `null`.

## Examples

### Example 1: Convert a number to single precision

```powerquery
Single.From(3.14159265358979)
```

    AsSingle = Single.From(Original)
