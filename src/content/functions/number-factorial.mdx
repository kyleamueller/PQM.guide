## Remarks

`Number.Factorial` computes n! — the product of all positive integers from 1 to n. By convention, `0! = 1`. The result grows extremely rapidly; `Number.Factorial(170)` is near the limit of double-precision floating point, and values beyond that will return infinity.

Factorials are fundamental to combinatorics. They underlie `Number.Combinations` (C(n,k) = n! / (k!(n-k)!)) and `Number.Permutations` (P(n,k) = n! / (n-k)!).

### Example 3: Manually verify combinations formula C(5,2)

C(5,2) = 5! / (2! * 3!) = 120 / (2 * 6) = 10

```powerquery
let
    N = 5,
    K = 2,
    Result = Number.Factorial(N) / (Number.Factorial(K) * Number.Factorial(N - K))
in
    Result
```
