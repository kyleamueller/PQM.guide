## Remarks

`Number.RoundTowardZero` truncates the fractional portion of a number, always moving the result closer to zero. For positive numbers this is equivalent to `Number.RoundDown`; for negative numbers it is equivalent to `Number.RoundUp` (toward zero, not toward negative infinity).

Unlike `Number.Round`, no tie-breaking rule applies — the fractional part is simply discarded regardless of its value.

This function is useful when you want strict truncation behavior: keeping only complete units and never inflating values by rounding up.

## Examples

### Example 1: Truncate a positive number

```powerquery
Number.RoundTowardZero(3.9)
```

### Example 4: Compute floor-divided age groups

Input: `Employees`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Employees, 4), {"FullName", "Age"}),
    "Decade", each Number.RoundTowardZero([Age] / 10) * 10, type number
)
```
