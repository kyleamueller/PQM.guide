## Remarks

`Guid.From` parses and normalizes a text value into a standard GUID (Globally Unique Identifier) format. The input may be in various common GUID formats, including with or without hyphens, with or without braces, and in any case (upper or lower). The output is always normalized to lowercase with hyphens.

If `value` is `null`, the function returns `null`. If `value` is not a valid GUID representation, an error is raised.

This function is useful for cleaning and standardizing GUID values loaded from external sources that may store them inconsistently (e.g., all caps, no hyphens, with curly braces).

Accepted input formats include:
- `"xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"` (32 hex digits, no hyphens)
- `"xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"` (standard with hyphens)
- `"{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}"` (with curly braces)

## Examples

### Example 1: Normalize a lowercase GUID with hyphens

```powerquery
Guid.From("a3f8c2d1-4b7e-4f2a-9c6d-1e8b3a5f0d72")
```
