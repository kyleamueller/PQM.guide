## Remarks

`Date.IsInPreviousDay` returns `true` if the input date falls on the calendar day immediately preceding today's date (`DateTime.LocalNow() - 1 day`). It does not include today itself. The function is re-evaluated on each query refresh, making it useful for identifying yesterday's transactions or events.

## Examples

### Example 1: Filter yesterday's orders

Keep only rows from the Sales table where OrderDate was yesterday.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInPreviousDay([OrderDate])
)
```

### Example 2: Flag yesterday's rows

Add a boolean column identifying rows from yesterday.

```powerquery
Table.AddColumn(
    Sales,
    "IsYesterday", each Date.IsInPreviousDay([OrderDate]), type logical
)
```
