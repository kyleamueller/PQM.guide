## Remarks

`Number.IntegerDivide` returns the integer portion of a division operation, discarding the remainder. It is equivalent to applying `Number.RoundDown` to the result of regular division. This makes it useful for grouping, bucketing, or computing how many whole units fit into a quantity.

For the corresponding remainder, pair this function with `Number.Mod`.

Dividing by zero raises an error. Use `try`/`otherwise` to handle potential division-by-zero scenarios in production queries.

## Examples

### Example 1: How many full weeks in a number of days

```powerquery
Number.IntegerDivide(17, 7)
```

<!--output
{"columns":[{"name":"Result","type":"number"}],"rows":[{"Result":2}]}

### Example 2: Compute full weeks and remaining days

```powerquery
let
    TotalDays = 17,
    Weeks = Number.IntegerDivide(TotalDays, 7),
    Remaining = Number.Mod(TotalDays, 7)
in
    #table({"Weeks", "RemainderDays"}, {{Weeks, Remaining}})
```

<!--output
{"columns":[{"name":"Weeks","type":"number"},{"name":"RemainderDays","type":"number"}],"rows":[{"Weeks":2,"RemainderDays":3}]}
-->

### Example 3: Bucket quantities into groups of 10

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Sales, 4), {"OrderID", "Quantity"}),
    "Bucket", each Number.IntegerDivide([Quantity], 10) * 10, type number
)
```
