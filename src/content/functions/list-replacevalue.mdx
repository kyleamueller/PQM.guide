## Remarks

`List.ReplaceValue` replaces items in a list using a replacer function. The `replacer` parameter controls how the comparison and substitution are performed:

- `Replacer.ReplaceValue` — performs an exact equality match and replaces matching items with `newValue`.
- `Replacer.ReplaceText` — performs a text substring replacement, replacing occurrences of `oldValue` (a text substring) within each text item with `newValue`.

This function is the list counterpart of the similarly-named table column replacement functions.

- All occurrences of `oldValue` are replaced.
- Items that don't match are passed through unchanged.
- When using `Replacer.ReplaceText`, each item must be a text value.

## Examples

### Example 1: Replace a specific value using ReplaceValue

```powerquery
List.ReplaceValue({1, 2, 3, 2, 4}, 2, 99, Replacer.ReplaceValue)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{1, 99, 3, 99, 4}"}]}
-->

### Example 2: Replace null values with zero

```powerquery
List.ReplaceValue({10, null, 30, null}, null, 0, Replacer.ReplaceValue)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{10, 0, 30, 0}"}]}
-->

### Example 3: Replace text substring within each item using ReplaceText

```powerquery
List.ReplaceValue({"Hello World", "World Peace", "New World"}, "World", "Earth", Replacer.ReplaceText)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{Hello Earth, Earth Peace, New Earth}"}]}
-->
