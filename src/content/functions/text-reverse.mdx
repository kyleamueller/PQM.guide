## Remarks

`Text.Reverse` returns a new text value containing all the characters of `text` in reverse order. If `text` is `null`, the function returns `null`. An empty string returns an empty string.

This function operates on individual Unicode code units. For text containing surrogate pairs or combining characters, results may not be linguistically meaningful.

Practical uses include palindrome detection, parsing right-to-left fixed-format strings, or reversing a portion of a text value when combined with `Text.Middle`.

## Examples

### Example 1: Reverse a simple string

```powerquery

```
