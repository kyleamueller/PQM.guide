## Remarks

`List.Contains` checks whether a specific value exists in a list. This is commonly used inside `Table.SelectRows` to filter rows where a column value is in a predefined set of values â€” similar to SQL's `IN` operator.

## Examples

### Example 1: Filter by a list of regions

Select only sales from East or North regions using List.Contains.

Input: `Sales`

```powerquery
let
    TargetRegions = {"East", "North"}
in
    Table.SelectColumns(
        Table.SelectRows(Sales, each List.Contains(TargetRegions, [Region])),
        {"OrderID", "CustomerName", "Region"}
    )
```
