## Remarks

`List.First` returns the first item from a list. Provide a default value to handle empty lists gracefully instead of raising an error. This is commonly used inside `Table.Group` aggregations â€” for example, `each List.First([Product])` to get the first product in each group.

## Examples

### Example 1: Get the first customer name

Return the first customer name from the Sales table.

Input: `Sales`

```powerquery
let
    FirstCustomer = List.First(Table.Column(Sales, "CustomerName"))
in
    #table({"FirstCustomer"}, {{FirstCustomer}})
```
