## Remarks

`Value.Equals` compares two values for equality using M's default equality semantics. It is equivalent to the `=` operator for most primitive types. Unlike the `=` operator however, `Value.Equals` can accept a custom comparer function via the optional third parameter, allowing case-insensitive text comparison, locale-aware comparison, or other custom equality logic.

Note that `null = null` in M evaluates to `null` (three-valued logic), whereas `Value.Equals(null, null)` returns `true`. Use `Value.NullableEquals` when you need three-valued null handling.
