## Remarks

`List.LastN` returns a **list** of items from the end of the source list. It accepts either a number or a condition function:

- **Number**: Returns the last `N` items. If `N` exceeds the list length, the entire list is returned.
- **Condition function**: Returns all consecutive items from the **end** of the list for which the condition is `true`, scanning backward.

When using a condition, the function scans from the end and stops at the first item that fails the condition — items before that point are excluded even if they would satisfy the condition.

## Examples

### Example 1: Return the last 3 items

```powerquery
List.LastN({10, 20, 30, 40, 50}, 3)
```

### Example 2: Return items from the end while value is greater than 25

```powerquery
List.LastN({10, 20, 30, 40, 50}, each _ > 25)
```

### Example 3: Get last week of dates from a date list

```powerquery
let
    Dates = List.Dates(#date(2024, 1, 1), 14, #duration(1, 0, 0, 0)),
    LastWeek = List.LastN(Dates, 7)
in
    LastWeek
```
