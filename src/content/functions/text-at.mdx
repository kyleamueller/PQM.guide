## Remarks

`Text.At` returns a single character from `text` at the specified zero-based `index`. The first character is at index 0, the second at index 1, and so on.

If `text` is `null`, the function returns `null`. If `index` is out of range (negative or equal to or greater than the length of the text), an error is raised.

This function behaves similarly to accessing a list item by index. It is equivalent to `Text.Middle(text, index, 1)`.

To extract multiple characters or a range, use `Text.Middle` or `Text.Range`. To get all characters as a list, use `Text.ToList`.
