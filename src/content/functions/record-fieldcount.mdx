## Remarks

`Record.FieldCount` returns the total number of fields in a record. This is equivalent to `List.Count(Record.FieldNames(record))`.

It is useful for validation — checking that a record has the expected number of fields before processing — and for dynamic iteration when you need to enumerate fields without knowing them in advance.

## Examples

### Example 1: Count fields in a simple record

```powerquery
Record.FieldCount([Name = "Alice", Age = 30, City = "New York"])
```

<!--output

### Example 2: Count fields in a row from the Customers table

Input: `Customers`

```powerquery
Record.FieldCount(Customers{0})
```

<!--output
{"columns":[{"name":"Result","type":"number"}],"rows":[{"Result":5}]}

### Example 3: Filter rows where a nested record has more than 3 fields

```powerquery
let
    Records = {
        [A=1, B=2, C=3],
        [A=1, B=2, C=3, D=4],
        [A=1, B=2]
    },
    AsTable = Table.FromRecords(Records),
    WithCount = Table.AddColumn(AsTable, "FieldCount", each Record.FieldCount(_), type number)
in
    WithCount
```
