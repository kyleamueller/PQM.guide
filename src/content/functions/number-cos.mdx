## Remarks

`Number.Cos` returns the trigonometric cosine of an angle measured in radians. The result is always in the range [-1, 1].

To convert degrees to radians, multiply by `Number.PI / 180`:

```powerquery
Number.Cos(degrees * Number.PI / 180)
```

`Number.Cos(0)` returns `1` and `Number.Cos(Number.PI)` returns `-1`.

## Examples

### Example 1: Cosine of 0 radians

```powerquery
Number.Cos(0)
```
