## Remarks

`Date.ToRecord` decomposes a `date` value into a record with three numeric fields: `Year`, `Month`, and `Day`. This is useful when you need all three components at once and want to avoid calling `Date.Year`, `Date.Month`, and `Date.Day` separately. You can expand the record into table columns using `Table.ExpandRecordColumn`.

## Examples

### Example 1: Expand date components into separate columns

Decompose each OrderDate into Year, Month, and Day columns.

Input: `Sales`

```powerquery
Table.ExpandRecordColumn(
    Table.AddColumn(
        Table.SelectColumns(
            Table.FirstN(Sales, 4),
            {"OrderID", "OrderDate"}
        ),
        "DateParts", each Date.ToRecord([OrderDate])
    ),
    "DateParts",
    {"Year", "Month", "Day"}
)
```

### Example 2: Decompose a specific date into a record

Return the Year, Month, Day record for March 15, 2024.

```powerquery
Date.ToRecord(#date(2024, 3, 15))
```
