## Remarks

`DateTime.Time` extracts the time component from a `datetime` or `datetimezone` value, discarding the date portion. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Extract time from OrderLog Timestamp

Extract the time part from each Timestamp in the OrderLog table.

Input: `OrderLog`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(OrderLog, 5),
        {"LogID", "Timestamp"}
    ),
    "TimeOnly", each DateTime.Time([Timestamp]), type time
)
```

<!--output
{"columns":[{"name":"LogID","type":"text"},{"name":"Timestamp","type":"datetime"},{"name":"TimeOnly","type":"time"}],"rows":[{"LogID":"L001","Timestamp":"2024-01-15T09:30:00","TimeOnly":"09:30:00"},{"LogID":"L002","Timestamp":"2024-01-16T14:15:00","TimeOnly":"14:15:00"},{"LogID":"L003","Timestamp":"2024-01-18T11:00:00","TimeOnly":"11:00:00"},{"LogID":"L004","Timestamp":"2024-01-20T08:45:00","TimeOnly":"08:45:00"},{"LogID":"L005","Timestamp":"2024-02-01T16:20:00","TimeOnly":"16:20:00"}]}
-->

### Example 2: Extract time from a scalar datetime

Return the time portion of a single datetime value.

```powerquery
#table(
    type table [DateTime = datetime, TimeOnly = time],
    {{#datetime(2024, 3, 15, 14, 30, 45), DateTime.Time(#datetime(2024, 3, 15, 14, 30, 45))}}
)
```

<!--output
{"columns":[{"name":"DateTime","type":"datetime"},{"name":"TimeOnly","type":"time"}],"rows":[{"DateTime":"2024-03-15T14:30:45","TimeOnly":"14:30:45"}]}
-->
