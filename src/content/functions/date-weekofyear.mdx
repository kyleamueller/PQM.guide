## Remarks

`Date.WeekOfYear` returns the week number within the calendar year, starting from 1. The first week is the one containing January 1. The `firstDayOfWeek` parameter controls which day begins a new week (default Sunday). Note that this uses a simple week counting convention and may differ from ISO 8601 week numbers, which define the first week as the one containing the first Thursday of the year. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Add week number to Sales table

Add a column showing the week number within the year for each order.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "WeekOfYear", each Date.WeekOfYear([OrderDate]), Int64.Type
)
```

### Example 2: Week of year with Monday start

Return the week number for March 15, 2024 using Monday as the first day of the week.

```powerquery
#table(
    type table [Date = date, WeekOfYear = number],
    {{#date(2024, 3, 15), Date.WeekOfYear(#date(2024, 3, 15), Day.Monday)}}
)
```
