## Remarks

`List.Mode` returns the single value that appears most frequently in the list. If multiple values are tied for the highest frequency (multiple modes), it returns the first one encountered in the original list order.

To retrieve **all** modes in a tie situation, use `List.Modes` instead.

- Null values are counted as occurrences and can be returned as the mode.
- Comparison is case-sensitive for text by default.
- Returns a single scalar value, not a list.
- An empty list causes an error.

## Examples

### Example 1: Single clear mode

```powerquery
List.Mode({1, 2, 2, 3, 3, 3, 4})
```

<!--output

### Example 2: Mode of text values

```powerquery
List.Mode({"A", "B", "A", "C", "B", "A"})
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"A"}]}
-->

### Example 3: Most common category in a dataset

```powerquery
let
    Categories = {"Electronics", "Clothing", "Electronics", "Books", "Clothing", "Electronics"},
    MostCommon = List.Mode(Categories)
in
    MostCommon
```
