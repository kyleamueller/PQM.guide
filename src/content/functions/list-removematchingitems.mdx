## Remarks

`List.RemoveMatchingItems` removes **all occurrences** of any value in `list2` from `list1`. Unlike `List.Difference` (which is a set operation returning distinct values), this function preserves duplicates in `list1` that are not in `list2`.

The key distinction from `List.RemoveItems`:
- `List.RemoveMatchingItems` removes **every** occurrence of matching values.
- `List.RemoveItems` removes one occurrence per matched item.

- Comparison is case-sensitive for text by default.
- Pass `Comparer.OrdinalIgnoreCase` for case-insensitive removal.
- Returns a new list; the original is not modified.

## Examples

### Example 1: Remove all matching values

```powerquery
List.RemoveMatchingItems({1, 2, 3, 2, 4, 2, 5}, {2, 4})
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{1, 3, 5}"}]}
-->

### Example 2: Remove stopwords from a word list

```powerquery
let
    Words = {"the", "quick", "the", "brown", "fox", "the"},
    Stopwords = {"the", "a", "an"},
    Filtered = List.RemoveMatchingItems(Words, Stopwords)
in
    Filtered
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{quick, brown, fox}"}]}
-->

### Example 3: Case-insensitive removal

```powerquery
List.RemoveMatchingItems({"Apple", "APPLE", "Banana", "apple"}, {"apple"}, Comparer.OrdinalIgnoreCase)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{Banana}"}]}
-->
