## Remarks

`Value.RemoveMetadata` strips metadata from a value. When called without the optional parameter, it removes all metadata. When a `metadataRecord` is provided, only the fields whose names appear in that record are removed from the metadata.

This is useful for cleaning up values before passing them to systems that should not receive internal metadata annotations, or for stripping specific internal bookkeeping fields while retaining others.

## Examples

### Example 1: Remove all metadata from a value

```powerquery
let
    Annotated = 42 meta [Source = "Raw", Verified = false],
    Clean = Value.RemoveMetadata(Annotated)
in
    Value.Metadata(Clean)
```

<!--output

### Example 2: The underlying value is preserved after removing metadata

```powerquery
let
    Annotated = 42 meta [Source = "Raw"],
    Clean = Value.RemoveMetadata(Annotated)
in
    Clean
```
