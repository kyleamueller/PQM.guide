## Remarks

`Int16.From` converts a value to a signed 16-bit integer (short integer), covering the range -32,768 to 32,767. Values outside this range raise an error. Fractional numbers are rounded using the specified rounding mode (default: round-toward-even).

Text values are parsed numerically. `true` converts to `1`, `false` to `0`, `null` to `null`.

This type maps to the `smallint` type in many databases and is useful for compact integer storage where values are known to fit within 16 bits.
