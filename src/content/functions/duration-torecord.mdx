## Remarks

`Duration.ToRecord` decomposes a `duration` value into a record with four numeric fields: `Days`, `Hours`, `Minutes`, and `Seconds`. Each field represents the corresponding component of the duration (not cumulative totals). For example, a duration of 1 day and 2.5 hours returns `[Days=1, Hours=2, Minutes=30, Seconds=0]`. Use `Table.ExpandRecordColumn` to expand the record into separate table columns.

## Examples

### Example 1: Expand duration components into separate columns

Decompose duration values in the Logs table into Days, Hours, Minutes, Seconds columns.

Input: `Logs`

```powerquery
Table.ExpandRecordColumn(
    Table.AddColumn(
        Table.SelectColumns(
            Table.SelectRows(Logs, each [DurationMinutes] <> null),
            {"LogID", "DurationMinutes"}
        ),
        "Parts",
        each Duration.ToRecord(#duration(0, 0, [DurationMinutes], 0))
    ),
    "Parts",
    {"Days", "Hours", "Minutes", "Seconds"}
)
```

### Example 2: Decompose a specific duration

Return the component record for a literal duration of 3 days, 5 hours, 30 minutes.

```powerquery
Duration.ToRecord(#duration(3, 5, 30, 0))
```
