## Remarks

`Type.Facets` returns a record describing the constraint properties (facets) of a type. Facets encode additional type metadata beyond the base kind — for example, `Precision` and `Scale` for decimal types, or `MaxLength` for text types.

For most primitive types with no additional constraints, the facets record will contain null values for all facet fields. Facets are most relevant when working with types surfaced from data sources that expose schema metadata (such as SQL Server columns with explicit precision and scale).

The complement of this function is `Type.ReplaceFacets`, which returns a new type with updated facets.

## Examples

### Example 1: Facets of a plain number type

```powerquery
Type.Facets(type number)
```
