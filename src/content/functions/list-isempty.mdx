## Remarks

`List.IsEmpty` returns `true` if the list has no items — i.e., when `List.Count(list) = 0`. It is a concise alternative to checking the count manually.

This function is useful for null-safe branching logic, such as checking whether a filter returned any results before accessing the first item.

- A list containing only null values is **not** empty — it has elements.
- Equivalent to `List.Count(list) = 0`.

## Examples

### Example 1: Empty list

```powerquery
List.IsEmpty({})
```

### Example 2: Non-empty list

### Example 3: Check if a filter returned results before accessing

```powerquery
let
    Values = {10, 20, 30},
    Filtered = List.Select(Values, each _ > 50),
    Result = if List.IsEmpty(Filtered) then null else List.First(Filtered)
in
    Result
```
