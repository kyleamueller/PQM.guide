## Remarks

`List.PositionOfAny` searches a list for the position of the first item that matches **any** of the values provided in the `values` list. It is a multi-target version of `List.PositionOf`.

- Returns `-1` if none of the values in `values` are found in `list`.
- Use `Occurrence.First` (default) to find the earliest match.
- Use `Occurrence.Last` to find the last item from `list` that matches any value.
- Use `Occurrence.All` to return a list of all positions where any of the target values appear.
- Comparison is case-sensitive for text by default.

## Examples

### Example 1: Find position of first match from a set of values

```powerquery
List.PositionOfAny({"A", "B", "C", "D", "E"}, {"C", "E"})
```

### Example 2: No match found

### Example 3: Find all positions of any matching value

```powerquery
List.PositionOfAny({"A", "B", "C", "A", "D", "B"}, {"A", "B"}, Occurrence.All)
```
