## Remarks

`Date.IsInNextWeek` returns `true` if the input date falls within the Sunday–Saturday calendar week immediately following the current week. The current week is not included; use `Date.IsInCurrentWeek` to also match the current week. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter next week's scheduled tasks

Keep only rows where TaskDate falls in the next calendar week.

```powerquery
Table.SelectRows(
    Tasks,
    each Date.IsInNextWeek([TaskDate])
)
```

### Example 2: Flag next-week items

Add a boolean column for tasks scheduled next week.

```powerquery
Table.AddColumn(
    Tasks,
    "NextWeek", each Date.IsInNextWeek([TaskDate]), type logical
)
```
