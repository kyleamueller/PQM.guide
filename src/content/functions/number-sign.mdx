## Remarks

`Number.Sign` returns `1` for any positive number, `-1` for any negative number, and `0` for zero. If the input is `null`, it returns `null`. This function is useful for directional classification — for example, labeling rows as gains, losses, or flat, or for applying signed operations without branching logic.

## Examples

### Example 1: Sign of a positive number

```powerquery
Number.Sign(42)
```

### Example 4: Classify profit/loss rows in Sales

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Sales, 4), {"OrderID", "Profit"}),
    "Direction",
    each if Number.Sign([Profit]) = 1 then "Gain"
         else if Number.Sign([Profit]) = -1 then "Loss"
         else "Flat",
    type text
)
```
