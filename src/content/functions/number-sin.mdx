## Remarks

`Number.Sin` returns the trigonometric sine of an angle measured in radians. The result is always in the range [-1, 1].

To convert degrees to radians before calling this function, multiply by `Number.PI / 180`:

```powerquery
Number.Sin(degrees * Number.PI / 180)
```

`Number.Sin(0)` returns `0`, `Number.Sin(Number.PI / 2)` returns `1`, and `Number.Sin(Number.PI)` returns approximately `0` (floating-point rounding may produce a very small non-zero value).
