## Remarks

`Table.PrefixColumns` renames every column in a table by adding a prefix followed by a period (e.g., `"Sales"` turns `"Product"` into `"Sales.Product"`). This is useful before combining or joining tables to avoid column name collisions â€” particularly when two tables share the same column names.

## Examples

### Example 1: Prefix column names

Prefix columns in a subset of the Sales table with "Sales".

Input: `Sales`

```powerquery
Table.PrefixColumns(
    Table.SelectColumns(Table.FirstN(Sales, 2), {"OrderID", "Product", "Region"}),
    "Sales"
)
```
