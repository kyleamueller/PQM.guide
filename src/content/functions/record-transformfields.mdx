## Remarks

`Record.TransformFields` applies functions to specified fields within a record. Each operation in the `transformOperations` list is either:

- A two-element list `{fieldName, function}` — transforms the named field in place
- A three-element list `{fieldName, function, newFieldName}` — transforms and renames the field

The transform function receives the current field value and returns the new value. Fields not listed in `transformOperations` are returned unchanged.

This function is the record-level equivalent of `Table.TransformColumns` for tables.

## Examples

### Example 1: Uppercase a text field

```powerquery
Record.TransformFields(
    [Name = "alice", Age = 30],
    {{"Name", Text.Upper}}
)
```

<!--output
{"columns":[{"name":"Name","type":"text"},{"name":"Age","type":"number"}],"rows":[{"Name":"ALICE","Age":30}]}
-->

### Example 2: Transform multiple fields

```powerquery
Record.TransformFields(
    [Name = "bob", Age = 25.7],
    {
        {"Name", Text.Proper},
        {"Age", Number.RoundDown}
    }
)
```

<!--output
{"columns":[{"name":"Name","type":"text"},{"name":"Age","type":"number"}],"rows":[{"Name":"Bob","Age":25}]}
-->

### Example 3: Transform and rename a field

```powerquery
Record.TransformFields(
    [price = 9.99, qty = 3],
    {{"price", each _ * 1.1, "PriceWithTax"}}
)
```

<!--output
{"columns":[{"name":"PriceWithTax","type":"number"},{"name":"qty","type":"number"}],"rows":[{"PriceWithTax":10.989000000000001,"qty":3}]}
-->
