## Remarks

`List.AnyTrue` returns `true` if at least one item in the list is logically true. It short-circuits and returns `true` as soon as a true value is found. Null values are treated as false.

This function pairs well with `List.Transform` to check whether any item in a list satisfies a given condition.

- An empty list returns `false`.
- Null values are treated as false.
- Non-logical values may cause an error; transform the list first if needed.

## Examples

### Example 1: At least one value is true

```powerquery
List.AnyTrue({false, false, true})
```

### Example 3: Check if any order exceeds a threshold

```powerquery
let
    Orders = {120, 85, 45, 200, 60},
    AnyLarge = List.AnyTrue(List.Transform(Orders, each _ > 150))
in
    AnyLarge
```
