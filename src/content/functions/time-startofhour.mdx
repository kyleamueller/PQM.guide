## Remarks

`Time.StartOfHour` returns the first instant of the clock hour containing the input value: XX:00:00. When passed a plain `time`, it returns a `time` with minutes and seconds set to zero. When passed a `datetime` or `datetimezone`, it returns the same type with only the time component adjusted. This is useful for bucketing events into hourly groups or building hourly range filters.

## Examples

### Example 1: Truncate event times to the hour

Round each event time down to the start of its hour.

Input: `Events`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Events, 4), {"EventID", "EventTime"}),
    "HourStart", each Time.StartOfHour([EventTime]), type datetime
)
```

### Example 2: Start of hour for a specific time

Return the first instant of the 14:xx hour.

```powerquery
#table(
    type table [Result = time],
    {{Time.StartOfHour(#time(14, 30, 45))}}
)
```

<!--output
{"columns":[{"name":"Result","type":"time"}],"rows":[{"Result":"14:00:00"}]}
