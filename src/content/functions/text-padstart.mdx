## Remarks

`Text.PadStart` adds `character` characters to the left (start) of `text` until the total length is at least `count`. If `text` is already `count` characters or longer, it is returned unchanged.

If `character` is omitted or `null`, a space (`" "`) is used. The `character` argument must be exactly one character long; providing a multi-character string raises an error.

If `text` is `null`, the function returns `null`.

Left-padding is commonly used to zero-pad numeric strings, align fixed-width report columns, or format codes to a consistent length.

## Examples

### Example 1: Zero-pad a number to five digits

```powerquery
Text.PadStart("42", 5, "0")
```
