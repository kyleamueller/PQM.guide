## Remarks

`Date.IsInNextNQuarters` returns `true` if the input date falls within the next N complete calendar quarters after the current quarter. The current quarter itself is not included; use `Date.IsInCurrentQuarter` to also match the current quarter. The function is re-evaluated on each query refresh.

For example, if today is in Q1 and `quarters = 2`, it matches any date in Q2 or Q3.

## Examples

### Example 1: Find projects due in the next 2 quarters

Keep only rows where ProjectEnd falls within the next 2 quarters.

```powerquery
Table.SelectRows(
    Projects,
    each Date.IsInNextNQuarters([ProjectEnd], 2)
)
```

### Example 2: Flag projects ending in the next 4 quarters

Add a boolean column for projects ending within the next year (4 quarters).

```powerquery
Table.AddColumn(
    Projects,
    "EndsNextYear", each Date.IsInNextNQuarters([ProjectEnd], 4), type logical
)
```
