## Remarks

`List.InsertRange` returns a new list with the specified `values` inserted at the given zero-based `index`. The original list is not modified.

- Items at and after the insertion point are shifted to the right.
- An index of `0` inserts at the beginning.
- An index equal to the list length inserts at the end (equivalent to appending).
- If the index is out of range, an error is raised.
- To insert a single value, wrap it in a list: `{value}`.

## Examples

### Example 1: Insert values in the middle of a list

```powerquery
List.InsertRange({1, 2, 5, 6}, 2, {3, 4})
```

### Example 2: Insert at the beginning


### Example 3: Insert at the end

```powerquery

```
