## Remarks

`Table.ColumnNames` returns the column names of a table as a list of text values, preserving the column order. This is useful for dynamic operations where you need to reference columns programmatically â€” for example, selecting all columns except a few, or building dynamic unpivot operations.

A common pattern is combining `Table.ColumnNames` with `List.RemoveItems` to get all column names except certain ones, then passing the result to `Table.SelectColumns` or `Table.UnpivotOtherColumns`.

## Examples

### Example 1: List column names

Get the column names from the Sales table.

Input: `Sales`

```powerquery
let
    Names = Table.ColumnNames(Sales)
in
    #table({"ColumnName"}, List.Transform(Names, each {_}))
```
