## Remarks

`List.Numbers` generates an arithmetic sequence of numeric values. It is the numeric equivalent of `List.Dates` and related sequence generators.

- Default increment is `1` if omitted or null.
- Use negative increment values to generate decreasing sequences.
- Use decimal increments for fractional steps (e.g., `0.5` for half-unit increments).
- `count` must be a non-negative integer; `0` returns an empty list.
- For simple integer ranges, the `{0..N}` literal syntax may be more concise.

## Examples

### Example 1: Generate integers from 1 to 5

```powerquery
List.Numbers(1, 5)
```
