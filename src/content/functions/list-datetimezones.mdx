## Remarks

`List.DateTimeZones` generates a sequence of `datetimezone` values, which include both a datetime and a UTC offset. This is useful when working with timezone-aware time series data.

- The `step` parameter uses `#duration(days, hours, minutes, seconds)` to specify the interval.
- The timezone offset from `start` is preserved in all generated values.
- Use `DateTimeZone.ToLocal` or `DateTimeZone.ToUtc` to convert the generated values if needed.
- `count` must be a non-negative integer; 0 returns an empty list.

## Examples

### Example 1: Generate hourly timezone-aware datetimes (UTC+5:30)

```powerquery
List.DateTimeZones(#datetimezone(2024, 1, 1, 0, 0, 0, 5, 30), 4, #duration(0, 1, 0, 0))
```

<!--output
{"columns":[{"name":"Result","type":"datetimezone"}],"rows":[{"Result":"2024-01-01 00:00:00 +05:30"},{"Result":"2024-01-01 01:00:00 +05:30"},{"Result":"2024-01-01 02:00:00 +05:30"},{"Result":"2024-01-01 03:00:00 +05:30"}]}

### Example 2: Generate daily UTC timestamps

```powerquery
List.DateTimeZones(#datetimezone(2024, 6, 1, 0, 0, 0, 0, 0), 5, #duration(1, 0, 0, 0))
```
