## Remarks

`Number.Cosh` returns the hyperbolic cosine, defined as:

cosh(x) = (e^x + e^(−x)) / 2

The hyperbolic cosine is always >= 1 and has its minimum at x = 0 where it equals 1. It is an even function: `cosh(−x) = cosh(x)`. The catenary curve — the shape of a hanging chain — is described by cosh.

The fundamental hyperbolic identity is: cosh²(x) − sinh²(x) = 1.

    X = 3,
    Identity = Number.Cosh(X)^2 - Number.Sinh(X)^2
