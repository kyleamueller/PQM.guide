## Remarks

`Date.IsInNextNMonths` returns `true` if the input date falls within the next N complete calendar months after the current month. The current month itself is not included; use `Date.IsInCurrentMonth` to also match the current month. The function is re-evaluated on each query refresh.

For example, if today is in March and `months = 3`, it matches any date in April, May, or June.

## Examples

### Example 1: Find renewals due in the next 3 months

Keep only rows where RenewalDate falls within the next 3 calendar months.

```powerquery
Table.SelectRows(
    Subscriptions,
    each Date.IsInNextNMonths([RenewalDate], 3)
)
```

### Example 2: Flag renewals in the next 6 months

Add a boolean column for subscriptions renewing within the next 6 months.

```powerquery
Table.AddColumn(
    Subscriptions,
    "RenewsInNext6Months", each Date.IsInNextNMonths([RenewalDate], 6), type logical
)
```
