## Remarks

`Table.FromValue` converts a single value into a table. The behavior depends on the input type:

- **Scalar** (number, text, date, etc.): produces a one-row, one-column table with the column named `Value`.
- **List**: produces a one-column table with one row per list element.
- **Record**: produces a two-column table with field names and values.
- **Table**: returns the table as-is.

Use the `options` record with `DefaultColumnName` to rename the default column from `Value` to something more descriptive.

## Examples

### Example 1: Wrap a scalar into a table

```powerquery
Table.FromValue(42)
```

<!--output
{"columns":[{"name":"Value","type":"any"}],"rows":[{"Value":"42"}]}
-->

### Example 2: Wrap a list into a table

```powerquery
Table.FromValue({1, 2, 3})
```

<!--output
{"columns":[{"name":"Value","type":"any"}],"rows":[{"Value":"1"},{"Value":"2"},{"Value":"3"}]}
-->

### Example 3: Use a custom column name

```powerquery
Table.FromValue({10, 20, 30}, [DefaultColumnName = "Amount"])
```

<!--output
{"columns":[{"name":"Amount","type":"any"}],"rows":[{"Amount":"10"},{"Amount":"20"},{"Amount":"30"}]}
-->
