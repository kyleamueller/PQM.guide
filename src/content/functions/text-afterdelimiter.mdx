## Remarks

`Text.AfterDelimiter` returns all text that follows the specified `delimiter` occurrence. By default it uses the first occurrence (index 0).

The `index` parameter can be:
- A non-negative integer: the zero-based occurrence to use (0 = first, 1 = second, etc.)
- A list `{n, RelativePosition.FromEnd}`: count occurrences from the end of the string

If the delimiter is not found, an empty string is returned. If `text` is `null`, the function returns `null`.

This function is commonly used in the Power Query UI when splitting columns by delimiter using the "Extract" option.

## Examples

### Example 1: Text after the first hyphen

```powerquery
Text.AfterDelimiter("US-WA-Seattle", "-")
```

<!--output

### Example 2: Text after the second hyphen

```powerquery
Text.AfterDelimiter("US-WA-Seattle", "-", 1)
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"Seattle"}]}
-->

### Example 3: Text after the last hyphen (from end)

```powerquery
Text.AfterDelimiter("US-WA-Seattle", "-", {0, RelativePosition.FromEnd})
```

<!--output
{"columns":[{"name":"Result","type":"text"}],"rows":[{"Result":"Seattle"}]}
-->

### Example 4: Extract domain from email address

```powerquery
Text.AfterDelimiter("user@example.com", "@")
```
