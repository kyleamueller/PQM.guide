## Remarks

`Date.EndOfQuarter` returns the last instant of the calendar quarter (Q1: Jan–Mar, Q2: Apr–Jun, Q3: Jul–Sep, Q4: Oct–Dec) that contains the given `date`, `datetime`, or `datetimezone` value. When given a plain `date`, the result is a `datetime` at 23:59:59.9999999 on the last day of the quarter. The return type matches the input type for `datetime` and `datetimezone`.

## Examples

### Example 1: Find the end of quarter for each order date

Add a column showing the end of the quarter for each order.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "QuarterEnd", each Date.EndOfQuarter([OrderDate]), type datetime
)
```

### Example 2: End of Q3

Return the last instant of Q3 2024.

```powerquery

    type table [Date = date, EndOfQ3 = datetime],
    {{#date(2024, 8, 15), Date.EndOfQuarter(#date(2024, 8, 15))}}
)
```
