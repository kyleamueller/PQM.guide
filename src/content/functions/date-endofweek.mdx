## Remarks

`Date.EndOfWeek` returns the last instant (23:59:59.9999999) of the calendar week containing the input value. By default, weeks start on Sunday (Day.Sunday = 0), making Saturday the last day. Pass `Day.Monday` (1) to use ISO-style Monday-to-Sunday weeks.

The return type matches the input type: a plain `date` input produces a `datetime` result; a `datetimezone` input preserves the timezone offset.

## Examples

### Example 1: Find the end of the week for each order date (Sunday start)

Add a column with the last day of the week (Saturday) for each order.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "WeekEnd", each Date.EndOfWeek([OrderDate]), type datetime
)
```

{"columns":[{"name":"OrderID","type":"number"},{"name":"OrderDate","type":"date"},{"name":"WeekEnd","type":"datetime"}],"rows":[{"OrderID":1,"OrderDate":"2024-01-15","WeekEnd":"2024-01-20T23:59:59.9999999"},{"OrderID":2,"OrderDate":"2024-01-18","WeekEnd":"2024-01-20T23:59:59.9999999"},{"OrderID":3,"OrderDate":"2024-02-01","WeekEnd":"2024-02-03T23:59:59.9999999"},{"OrderID":4,"OrderDate":"2024-02-10","WeekEnd":"2024-02-10T23:59:59.9999999"},{"OrderID":5,"OrderDate":"2024-03-05","WeekEnd":"2024-03-09T23:59:59.9999999"}]}
-->

### Example 2: End of ISO week (Monday start)

Return the end of the ISO week (Sunday) for a specific date.

```powerquery
#table(
    type table [Date = date, ISOWeekEnd = datetime],
    {{#date(2024, 3, 13), Date.EndOfWeek(#date(2024, 3, 13), Day.Monday)}}
)
```

<!--output
{"columns":[{"name":"Date","type":"date"},{"name":"ISOWeekEnd","type":"datetime"}],"rows":[{"Date":"2024-03-13","ISOWeekEnd":"2024-03-17T23:59:59.9999999"}]}
-->
