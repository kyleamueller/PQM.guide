## Remarks

`Date.DayOfYear` returns an integer from 1 (January 1) to 365 or 366 (December 31, depending on leap year) for a `date`, `datetime`, or `datetimezone` value. This is useful for day-of-year calculations such as comparing progress through the year across different years. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Add day-of-year number to Sales table

Add a column showing the ordinal day of the year for each order date.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "DayOfYear", each Date.DayOfYear([OrderDate]), Int64.Type
)
```

### Example 2: Day of year for a specific date

Return the day-of-year number for March 15, 2024.

```powerquery
#table(
    type table [Date = date, DayOfYear = number],
    {{#date(2024, 3, 15), Date.DayOfYear(#date(2024, 3, 15))}}
)
```
