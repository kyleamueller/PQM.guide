## Remarks

`List.Dates` is the standard way to generate a continuous or stepped sequence of date values in Power Query M. It is commonly used to create date dimension tables, fill in missing date gaps, or generate a calendar list.

- The `step` parameter is a `duration` value. Use `#duration(days, 0, 0, 0)` for day-based steps.
- To generate weekly dates, use `#duration(7, 0, 0, 0)`.
- To generate monthly dates, use `Date.AddMonths` with `List.Generate` instead, since months have variable lengths.
- `count` must be a non-negative integer. Passing 0 returns an empty list.

## Examples

### Example 1: Generate a week of daily dates

```powerquery
List.Dates(#date(2024, 1, 1), 7, #duration(1, 0, 0, 0))
```


### Example 3: Build a date dimension table

Generate a full year of dates as a table.

```powerquery
let
    DateList = List.Dates(#date(2024, 1, 1), 366, #duration(1, 0, 0, 0)),
    DateTable = Table.FromList(DateList, Splitter.SplitByNothing(), {"Date"})
in
    DateTable
```
