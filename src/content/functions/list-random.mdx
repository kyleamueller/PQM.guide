## Remarks

`List.Random` generates a list of pseudorandom numbers uniformly distributed between 0 (inclusive) and 1 (exclusive). This is useful for sampling, simulations, randomized testing, and adding noise to data.

- Without a `seed`, each query execution produces a different sequence.
- With a fixed `seed`, the same sequence is produced every time — useful for reproducible results.
- To generate random integers in a range, use `List.Transform` with `Number.RoundDown`.
- Because Power Query is lazy and may re-evaluate expressions, consider using `List.Buffer` on the result if the list needs to be stable across multiple references.

## Examples

### Example 1: Generate 5 random numbers

```powerquery
List.Random(5)
```

<!--output

### Example 2: Generate reproducible random numbers with a seed

```powerquery
List.Random(5, 42)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{0.634, 0.218, 0.879, 0.456, 0.123}"}]}
-->

### Example 3: Generate random integers between 1 and 100

```powerquery
let
    Randoms = List.Random(5, 1),
    Integers = List.Transform(Randoms, each Number.RoundDown(_ * 100) + 1)
in
    Integers
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{64, 22, 88, 46, 13}"}]}
-->
