## Remarks

`Number.ToText` converts a number to a text string. When called with no format, it returns a general representation of the number. The optional `format` parameter follows standard .NET numeric format specifiers:

- `"D"` — Integer (no decimal places, e.g., `"42"`)
- `"F"` or `"F2"` — Fixed-point with specified decimal places (e.g., `"3.14"`)
- `"P"` — Percentage (e.g., `"75.00%"`)
- `"E"` — Scientific notation (e.g., `"3.14E+000"`)
- `"C"` — Currency symbol added based on culture
- `"G"` — General (default compact representation)
- `"N"` — Number with thousands separators

The `culture` parameter controls locale-specific symbols. For example, `"de-DE"` uses a comma as the decimal separator.

## Examples

### Example 1: Convert a number to fixed 2 decimal places

```powerquery
Number.ToText(3.14159, "F2")
```

### Example 3: Format numbers in a column with thousands separators

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Sales, 3), {"OrderID", "Revenue"}),
    "FormattedRevenue", each Number.ToText([Revenue], "N2"), type text
)
```
