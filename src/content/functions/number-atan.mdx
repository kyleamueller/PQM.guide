## Remarks

`Number.Atan` is the inverse of `Number.Tan`. Given any real number, it returns the angle in radians whose tangent equals that value. The result is always in the open interval (-π/2, π/2).

Unlike `Number.Acos` and `Number.Asin`, `Number.Atan` accepts any real number (not just values in [-1, 1]). For two-argument arctangent that correctly handles all four quadrants, use `Number.Atan2`.

To convert the result from radians to degrees, multiply by `180 / Number.PI`.

## Examples

### Example 1: Arctangent of 1 (45 degrees)

```powerquery
Number.Atan(1) * 180 / Number.PI
```
