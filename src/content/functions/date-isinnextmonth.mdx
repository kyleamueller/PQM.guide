## Remarks

`Date.IsInNextMonth` returns `true` if the input date falls within the calendar month immediately after the current month (as determined by `DateTime.LocalNow()`). For example, if today is in March, this function returns `true` for any date in April. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter next month's renewals

Keep only rows where RenewalDate falls in the next calendar month.

```powerquery
Table.SelectRows(
    Subscriptions,
    each Date.IsInNextMonth([RenewalDate])
)
```

### Example 2: Flag next-month rows

Add a boolean column identifying rows due next month.

```powerquery
Table.AddColumn(
    Subscriptions,
    "RenewsNextMonth", each Date.IsInNextMonth([RenewalDate]), type logical
)
```
