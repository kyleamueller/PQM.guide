## Remarks

`Byte.From` converts a value to a byte — an 8-bit unsigned integer in the range 0 to 255. Values outside this range raise an error. Fractional numbers are rounded according to the optional `roundingMode` parameter (defaulting to round-toward-even).

Text values are parsed as numbers before conversion. A logical `true` converts to `1`, `false` to `0`. `null` returns `null`.

This type is useful when working with binary data, raw byte buffers, or systems that require small fixed-range integer fields.
