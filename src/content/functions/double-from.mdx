## Remarks

`Double.From` converts a value to a double-precision floating-point number — the standard numeric type used throughout Power Query M. Double-precision provides approximately 15–17 significant decimal digits of precision and covers an enormous range (~±1.8 × 10^308).

Text values are parsed using the specified culture (or the session locale by default). `true` converts to `1.0`, `false` to `0.0`, `null` to `null`.

This is effectively equivalent to `Number.From` in most contexts, since the default M number type is double-precision. Use it explicitly when you need to annotate or enforce the type annotation in a query.

## Examples

### Example 1: Convert text to double

```powerquery
Double.From("3.14159")
```
