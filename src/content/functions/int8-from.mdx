## Remarks

`Int8.From` converts a value to a signed 8-bit integer, allowing values from -128 to 127. Values outside this range raise an error. Fractional numbers are rounded using the specified rounding mode, defaulting to round-toward-even (banker's rounding).

Text values are parsed numerically. `true` converts to `1`, `false` to `0`, `null` to `null`.

Use this type when interfacing with systems or binary formats that use compact signed byte fields.
