## Remarks

`Text.FromBinary` decodes a binary value into a text string using the specified character encoding. This is the inverse of `Text.ToBinary`.

The `encoding` parameter accepts values from the `TextEncoding` enum:
- `TextEncoding.Utf8` (65001) — default
- `TextEncoding.Ascii` (20127)
- `TextEncoding.Unicode` / `TextEncoding.Utf16` (1200)
- `TextEncoding.Utf16BE` (1201)
- `TextEncoding.Utf32` (12000)
- `TextEncoding.Windows` (1252)

Common uses include decoding base64-decoded binary payloads (e.g., from web APIs), reading binary file content as text, or processing encoded data returned by connectors.

## Examples

### Example 1: Decode UTF-8 binary to text

```powerquery
Text.FromBinary(Text.ToBinary("Hello World", TextEncoding.Utf8), TextEncoding.Utf8)
```
