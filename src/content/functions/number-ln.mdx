## Remarks

`Number.Ln` computes the natural logarithm — the logarithm to base _e_ (approximately 2.71828). It is the inverse of `Number.Exp`: `Number.Ln(Number.Exp(x)) = x`.

Inputs must be positive real numbers for a defined result. Passing `0` returns negative infinity; passing a negative number returns `NaN`.

Common uses include continuous compounding calculations, entropy, normalizing skewed data, and log-scale transformations.

## Examples

### Example 1: Natural log of e

```powerquery
Number.Ln(Number.E)
```
