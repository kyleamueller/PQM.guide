## Remarks

`DateTime.IsInNextHour` returns `true` if the input `datetime` or `datetimezone` value falls within the clock hour immediately following the current hour (as determined by `DateTime.LocalNow()`). For example, if it is currently 10:xx, this function returns `true` for any datetime between 11:00:00 and 11:59:59.9999999. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter events scheduled for the next hour

Keep only rows where EventTime falls in the next clock hour.

```powerquery
Table.SelectRows(
    Events,
    each DateTime.IsInNextHour([EventTime])
)
```

### Example 2: Flag next-hour events

Add a boolean column for events in the next hour.

```powerquery
Table.AddColumn(
    Events,
    "IsNextHour", each DateTime.IsInNextHour([EventTime]), type logical
)
```
