## Remarks

`Table.DuplicateColumn` creates a copy of an existing column with a new name. This is useful when you want to transform a column but also keep the original values â€” duplicate first, then transform the copy.

The new column is appended to the end of the table. Use `Table.ReorderColumns` if you need it in a specific position.

## Examples

### Example 1: Duplicate a column for transformation

Duplicate the Product column, then use the copy for further processing.

Input: `Sales`

```powerquery
Table.SelectColumns(
    Table.DuplicateColumn(Table.FirstN(Sales, 3), "Product", "OriginalProduct"),
    {"OrderID", "Product", "OriginalProduct"}
)
```
