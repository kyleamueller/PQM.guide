## Remarks

`List.Split` divides a list into consecutive chunks (pages), each containing at most `pageSize` items. If the total number of items is not evenly divisible by `pageSize`, the last sub-list will contain the remaining items (fewer than `pageSize`).

This function is useful for batching API calls, chunking large datasets for processing, or implementing pagination logic.

- The result is a **list of lists**, not a flat list.
- The `pageSize` must be a positive integer.
- If the source list is empty, the result is an empty list.
- All chunks except possibly the last will have exactly `pageSize` items.

## Examples

### Example 1: Split into chunks of 3

```powerquery
List.Split({1, 2, 3, 4, 5, 6, 7, 8, 9}, 3)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}"}]}
-->

### Example 2: Last chunk may be smaller

```powerquery
List.Split({1, 2, 3, 4, 5}, 2)
```

<!--output
{"columns":[{"name":"Result","type":"any"}],"rows":[{"Result":"{{1, 2}, {3, 4}, {5}}"}]}
-->

### Example 3: Process a list in batches of 100

```powerquery
let
    AllIDs = List.Numbers(1, 250),
    Batches = List.Split(AllIDs, 100),
    BatchCount = List.Count(Batches)
in
    BatchCount
```
