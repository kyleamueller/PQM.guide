## Remarks

`Replacer.ReplaceText` is a built-in replacer **function** used as an argument to `Table.ReplaceValue`. It performs **substring** replacement â€” it finds occurrences of `oldValue` within the text of each cell and replaces them with `newValue`, even if the cell contains additional text.

Use `Replacer.ReplaceText` for partial text replacements (e.g., replacing "Widget" with "Component" inside product names). For replacing entire cell values, use `Replacer.ReplaceValue` instead.

## Examples

### Example 1: Substring replacement in product names

Replace the substring "Widget" with "Component" in the Product column. This affects any product name containing "Widget" as part of its text.

Input: `Sales`

```powerquery
Table.SelectColumns(
    Table.ReplaceValue(
        Table.FirstN(Sales, 4),
        "Widget", "Component",
        Replacer.ReplaceText, {"Product"}
    ),
    {"CustomerName", "Product", "Category"}
)
```
