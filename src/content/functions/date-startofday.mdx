## Remarks

`Date.StartOfDay` returns the very first instant of the day (00:00:00) as a `datetime`. When passed a plain `date`, it returns a `datetime` with the time component set to midnight. When passed a `datetimezone`, it preserves the timezone offset. This is useful for building date range filters to capture all events that occurred on a particular day.

## Examples

### Example 1: Filter events from the start of a specific day

Keep only rows where EventTime is on or after the start of January 15, 2024.

Input: `Events`

```powerquery
Table.SelectRows(
    Events,
    each [EventTime] >= Date.StartOfDay(#date(2024, 1, 15))
)
```

<!--output
{"columns":[{"name":"EventID","type":"number"},{"name":"EventTime","type":"datetime"}],"rows":[{"EventID":1,"EventTime":"2024-01-15T08:30:00"},{"EventID":2,"EventTime":"2024-01-15T23:59:59"}]}
-->

### Example 2: Start of day for a specific date

Return midnight on March 15, 2024.

```powerquery
#table(
    type table [Result = datetime],
    {{Date.StartOfDay(#date(2024, 3, 15))}}
)
```

<!--output
{"columns":[{"name":"Result","type":"datetime"}],"rows":[{"Result":"2024-03-15T00:00:00"}]}
-->
