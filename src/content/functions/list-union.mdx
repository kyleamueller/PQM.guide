## Remarks

`List.Union` takes a **list of lists** as its argument and returns a single list containing all distinct values from all the input lists combined. It is equivalent to combining all lists with `List.Combine` and then deduplicating with `List.Distinct`.

- The argument is a list of lists: `{list1, list2, list3}`.
- The result contains only distinct values — duplicates across or within input lists are removed.
- Order of items in the result follows first appearance across the input lists in order.
- Comparison is case-sensitive for text by default. Pass `Comparer.OrdinalIgnoreCase` for case-insensitive deduplication.

## Examples

### Example 1: Union of two lists

```powerquery
List.Union({{1, 2, 3}, {2, 3, 4, 5}})
```

### Example 2: Union of three lists

```powerquery
List.Union({{"A", "B"}, {"B", "C"}, {"C", "D", "E"}})
```

<!--output

### Example 3: Combine customer lists from multiple regions

```powerquery
let
    RegionA = {"Cust-01", "Cust-02", "Cust-03"},
    RegionB = {"Cust-02", "Cust-04"},
    RegionC = {"Cust-03", "Cust-05"},
    AllCustomers = List.Union({RegionA, RegionB, RegionC})
in
    AllCustomers
```
