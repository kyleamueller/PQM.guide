## Remarks

`List.Product` multiplies all values in a numeric list together and returns the result. This is useful for compound growth calculations, factorial computations, and any scenario requiring sequential multiplication.

- Returns `null` if the list is empty.
- Null values in the list are skipped (treated as if they are not present).
- Including `0` in the list will produce a result of `0`.
- Use `Precision.Double` as the second argument to use IEEE 754 double-precision floating-point arithmetic, which may differ from the default decimal arithmetic for very large or very small numbers.

## Examples

### Example 1: Product of a simple list

```powerquery
List.Product({1, 2, 3, 4, 5})
```

### Example 2: Compound growth calculation

Calculate the cumulative growth factor over 4 years with annual growth rates.

```powerquery
let
    GrowthRates = {1.05, 1.03, 1.08, 1.02},
    CumulativeGrowth = List.Product(GrowthRates)
in
    CumulativeGrowth
```
