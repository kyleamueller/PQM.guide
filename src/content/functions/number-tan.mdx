## Remarks

`Number.Tan` returns the trigonometric tangent of an angle measured in radians. Tangent is defined as sine divided by cosine: tan(x) = sin(x) / cos(x).

The function is undefined at odd multiples of π/2 (90°, 270°, etc.), where cosine equals zero. At these points the result approaches positive or negative infinity. Floating-point representation may return a very large finite number rather than true infinity.

To convert degrees to radians, multiply by `Number.PI / 180`.

## Examples

### Example 1: Tangent of 0 radians

```powerquery
Number.Tan(0)
```
