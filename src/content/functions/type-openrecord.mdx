## Remarks

`Type.OpenRecord` converts a closed record type into an open record type. An open record type allows values to carry extra fields not declared in the type definition.

If the input is already an open record type, it is returned unchanged. This is the complement of `Type.ClosedRecord`.

Open record types are useful when you want to declare required fields for validation or type-checking while still permitting additional data to flow through without raising type errors.

## Examples

### Example 1: Open a closed record type

```powerquery
let
    Closed = Type.ClosedRecord(type record),
    Opened = Type.OpenRecord(Closed)
in
    Type.IsOpenRecord(Opened)
```

### Example 2: Open a typed record type with declared fields

```powerquery

    Closed = Type.ForRecord([Name = [Type = type text, Optional = false]], false),
    Opened = Type.OpenRecord(Closed)
```
