## Remarks

`Int64.From` converts a value to a signed 64-bit integer (long integer). This is the largest fixed-size integer type in Power Query M, covering approximately ±9.2 × 10^18. Fractional values are rounded using the optional rounding mode (default: round-toward-even).

Text values are parsed numerically. `true` converts to `1`, `false` to `0`, `null` to `null`.

In Power BI and Analysis Services, this type is called `Whole Number`. It maps to `bigint` in SQL Server. Use this for IDs, row counts, timestamps in epoch format, or any integer that may exceed the 32-bit range.

## Examples

### Example 1: Convert a large number

```powerquery
Int64.From(9007199254740992)
```
