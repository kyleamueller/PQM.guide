## Remarks

`List.Intersect` takes a **list of lists** as its first argument (not two separate lists) and returns a new list containing only items that are present in **every** provided list. The result contains distinct values only.

This function is useful for finding common values across multiple datasets — for example, customers who appear in all regions, or products available in every store.

- The argument is a list of lists: `{list1, list2, list3}`.
- The result is deduplicated (distinct values only).
- Comparison is case-sensitive for text by default. Pass `Comparer.OrdinalIgnoreCase` for case-insensitive matching.
- If any list in the input is empty, the result is an empty list.

## Examples

### Example 1: Intersect two lists

```powerquery
List.Intersect({{1, 2, 3, 4}, {2, 4, 6}})
```

### Example 3: Find common customers across regions

```powerquery
let
    RegionA = {"Cust-01", "Cust-02", "Cust-03"},
    RegionB = {"Cust-02", "Cust-03", "Cust-04"},
    RegionC = {"Cust-02", "Cust-05", "Cust-03"},
    CommonCustomers = List.Intersect({RegionA, RegionB, RegionC})
in
    CommonCustomers
```
