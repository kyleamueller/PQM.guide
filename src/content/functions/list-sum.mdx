## Remarks

`List.Sum` calculates the total of all numeric values in a list. Null values are ignored. This is commonly used inside `Table.Group` aggregations â€” for example, `each List.Sum([Quantity])` to sum quantities per group.

## Examples

### Example 1: Sum all quantities

Calculate the total quantity across all sales.

Input: `Sales`

```powerquery
let
    Total = List.Sum(Table.Column(Sales, "Quantity"))
in
    #table({"TotalQuantity"}, {{Total}})
```
