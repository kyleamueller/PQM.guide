## Remarks

`Date.IsInNextNYears` returns `true` if the input date falls within the next N complete calendar years after the current year. The current year itself is not included; use `Date.IsInCurrentYear` to also match the current year. The function is re-evaluated on each query refresh.

For example, if the current year is 2026 and `years = 3`, it matches any date in 2027, 2028, or 2029.

## Examples

### Example 1: Find contracts expiring in the next 2 years

Keep only rows where ExpiryDate falls within the next 2 calendar years.

```powerquery
Table.SelectRows(
    Contracts,
    each Date.IsInNextNYears([ExpiryDate], 2)
)
```

### Example 2: Flag long-term contracts

Add a boolean column for contracts expiring in the next 5 years.

```powerquery
Table.AddColumn(
    Contracts,
    "ExpiresInNext5Years", each Date.IsInNextNYears([ExpiryDate], 5), type logical
)
```
