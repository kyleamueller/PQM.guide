## Remarks

`Date.IsInPreviousNYears` returns `true` if the input date falls within the N complete calendar years preceding the current year. The current year itself is not included; use `Date.IsInCurrentYear` to also match the current year. The function is re-evaluated on each query refresh.

For example, if the current year is 2026 and `years = 3`, it matches any date in 2023, 2024, or 2025.

## Examples

### Example 1: Filter orders from the previous 3 years

Keep only rows where OrderDate falls within the previous 3 calendar years.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInPreviousNYears([OrderDate], 3)
)
```

### Example 2: Flag records from the past 5 years

Add a boolean column for records within the previous 5 years.

```powerquery
Table.AddColumn(
    HistoricalData,
    "IsRecentHistory", each Date.IsInPreviousNYears([RecordDate], 5), type logical
)
```
