## Remarks

`Number.Combinations` computes the binomial coefficient C(n, k), read as "n choose k". It returns the number of ways to select `combinationSize` items from a collection of `setSize` items when the order of selection does not matter.

The formula is: C(n, k) = n! / (k! × (n − k)!)

Common applications include probability calculations, lottery odds, sampling scenarios, and feature selection counts. When `combinationSize` equals 0 or `setSize`, the result is 1. When `combinationSize` > `setSize`, the result is 0.

## Examples

### Example 1: How many ways to choose 2 items from 5

```powerquery
Number.Combinations(5, 2)
```
