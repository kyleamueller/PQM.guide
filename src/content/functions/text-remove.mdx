## Remarks

`Text.Remove` strips all occurrences of one or more characters from a text value. Unlike `Text.Replace`, which replaces substrings, `Text.Remove` works with individual characters â€” pass a list of characters to remove multiple character types at once.

This is ideal for stripping non-numeric characters from phone numbers, removing punctuation, or cleaning formatting characters.

## Examples

### Example 1: Extract digits from phone numbers

Remove all non-digit characters from phone numbers.

Input: `Employees`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Employees, 3), {"Phone"}),
    "DigitsOnly", each Text.Remove([Phone], {"(", ")", " ", "-"}), type text
)
```
