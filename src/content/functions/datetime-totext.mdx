## Remarks

`DateTime.ToText` formats a `datetime` value as a text string. Without options, it produces an ISO 8601-style string (e.g., `"2024-03-15T14:30:00"`). Use the `Format` field in the options record to apply custom patterns such as `"dd/MM/yyyy HH:mm"` or `"MMMM d, yyyy h:mm tt"`. The `Culture` field controls locale-specific formatting such as localized month or day names. Format strings follow .NET custom datetime format conventions.

If the input is `null`, the function returns `null`.

## Examples

### Example 1: Format datetime values for a report label

Convert EventTime to a human-readable string for display.

Input: `Events`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Events, 4),
        {"EventID", "EventTime"}
    ),
    "DisplayTime",
    each DateTime.ToText([EventTime], [Format="MMMM d, yyyy h:mm tt"]),
    type text
)
```

-->

### Example 2: Format in 24-hour ISO style

Convert a specific datetime to a sortable ISO-format string.

```powerquery
#table(
    type table [DateTime = datetime, ISO = text],
    {{#datetime(2024, 3, 15, 14, 30, 0), DateTime.ToText(#datetime(2024, 3, 15, 14, 30, 0), [Format="yyyy-MM-dd HH:mm:ss"])}}
)
```

<!--output
{"columns":[{"name":"DateTime","type":"datetime"},{"name":"ISO","type":"text"}],"rows":[{"DateTime":"2024-03-15T14:30:00","ISO":"2024-03-15 14:30:00"}]}
-->
