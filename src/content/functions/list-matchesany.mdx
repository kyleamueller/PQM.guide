## Remarks

`List.MatchesAny` applies a condition function to each item in the list and returns `true` as soon as any item satisfies the condition. It is the list equivalent of the "there exists" quantifier.

Unlike `List.AnyTrue`, which expects a pre-computed list of logical values, `List.MatchesAny` accepts any list and a predicate function.

- An empty list returns `false`.
- The function short-circuits: evaluation stops as soon as a passing item is found.
- Equivalent to `List.AnyTrue(List.Transform(list, condition))`, but may short-circuit earlier.
- Use `List.Contains` for simple equality checks; `List.MatchesAny` is better for complex conditions.

## Examples

### Example 1: Check if any number exceeds a threshold

```powerquery
List.MatchesAny({10, 25, 5, 8}, each _ > 20)
```

### Example 3: Check if any product is out of stock

```powerquery
let
    StockLevels = {100, 0, 45, 230, 0},
    HasOutOfStock = List.MatchesAny(StockLevels, each _ = 0)
in
    HasOutOfStock
```
