## Remarks

`Text.BetweenDelimiters` extracts the portion of `text` that lies between a `startDelimiter` and an `endDelimiter`. The search for the end delimiter begins immediately after the start delimiter is found.

The optional `startIndex` and `endIndex` parameters control which occurrence of each delimiter to use:
- A non-negative integer specifies the zero-based occurrence (0 = first)
- A list `{n, RelativePosition.FromEnd}` counts occurrences from the end of the string

If either delimiter is not found, an empty string is returned. If `text` is `null`, the function returns `null`.

This function is especially useful for extracting content from structured formats such as XML-like tags, parenthesized values, or bracketed codes.

## Examples

### Example 1: Extract content between parentheses

```powerquery
Text.BetweenDelimiters("Product (SKU-123) Details", "(", ")")
```
