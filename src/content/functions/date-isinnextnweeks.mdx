## Remarks

`Date.IsInNextNWeeks` returns `true` if the input date falls within the next N complete calendar weeks (Sunday–Saturday) after the current week. The current week itself is not included; use `Date.IsInCurrentWeek` to also match the current week. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Find tasks due in the next 2 weeks

Keep only rows where DueDate falls within the next 2 calendar weeks.

```powerquery
Table.SelectRows(
    Tasks,
    each Date.IsInNextNWeeks([DueDate], 2)
)
```

### Example 2: Flag deliverables due in the next 4 weeks

Add a boolean column for items due within the next 4 weeks.

```powerquery
Table.AddColumn(
    Deliverables,
    "DueInNext4Weeks", each Date.IsInNextNWeeks([DueDate], 4), type logical
)
```
