## Remarks

`Date.IsInCurrentDay` compares the date component of the input value against today's date as determined by `DateTime.LocalNow()`. It returns `true` if they fall on the same calendar day, regardless of the time component. This function is re-evaluated on each query refresh, making it suitable for dynamic "today" filters.

All `Date.IsIn*` functions use the local system clock (`DateTime.LocalNow`) as their reference point. In Power BI Service scheduled refreshes, this is the time zone of the Power BI gateway or the cloud region.

## Examples

### Example 1: Filter today's orders

Keep only rows from the Sales table where OrderDate is today.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInCurrentDay([OrderDate])
)
```

### Example 2: Flag today's rows with a boolean column

Add an IsToday column to identify rows from today.

```powerquery
Table.AddColumn(
    Sales,
    "IsToday", each Date.IsInCurrentDay([OrderDate]), type logical
)
```
