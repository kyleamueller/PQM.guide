## Remarks

`List.RemoveFirstN` is equivalent to `List.Skip` — it removes the first `N` items from a list and returns the remainder. It accepts either a count or a condition function.

- **Number**: Removes the first `N` items. If `N` is larger than the list, an empty list is returned.
- **Condition function**: Removes items from the start of the list while the condition returns `true`, stopping at the first item that fails.
- This function does not modify the original list; it returns a new list.

## Examples

### Example 1: Remove the first 2 items

```powerquery
List.RemoveFirstN({1, 2, 3, 4, 5}, 2)
```

### Example 2: Remove items while value is less than 30

```powerquery
List.RemoveFirstN({10, 20, 30, 40, 50}, each _ < 30)
```

<!--output

### Example 3: Skip header-like rows (remove first 3)

```powerquery
let
    Rows = {"Header", "Subtitle", "---", "Data1", "Data2", "Data3"},
    DataOnly = List.RemoveFirstN(Rows, 3)
in
    DataOnly
```
