## Remarks

`Text.Range` returns a substring from `text` starting at the zero-based `offset` index. If `count` is provided, at most `count` characters are returned. If `count` is omitted or `null`, all characters from `offset` to the end of the string are returned.

If `text` is `null`, the function returns `null`. If `offset` is beyond the end of the string, an empty string is returned. If `count` extends beyond the available characters, all remaining characters are returned without error.

`Text.Range` and `Text.Middle` are functionally equivalent — both accept an offset and optional count. Use whichever is clearer in context.

