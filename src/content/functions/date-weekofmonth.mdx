## Remarks

`Date.WeekOfMonth` returns the week number within the month, starting from 1. The `firstDayOfWeek` parameter controls which day starts a new week (default Sunday). Week 1 contains the first day of the month; partial weeks at month boundaries are still counted. If the input is `null`, the function returns `null`.

## Examples

### Example 1: Add week-of-month to Sales table

Add a column showing the week number within the month for each order.

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(
        Table.FirstN(Sales, 5),
        {"OrderID", "OrderDate"}
    ),
    "WeekOfMonth", each Date.WeekOfMonth([OrderDate]), Int64.Type
)
```

### Example 2: Week of month using Monday as start of week

Return the week-of-month for March 15, 2024 with Monday as the first day.

```powerquery
#table(
    type table [Date = date, WeekOfMonth = number],
    {{#date(2024, 3, 15), Date.WeekOfMonth(#date(2024, 3, 15), Day.Monday)}}
)
```
