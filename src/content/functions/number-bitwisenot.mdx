## Remarks

`Number.BitwiseNot` flips every bit in the integer representation of the input. For a signed 64-bit integer representation, the result is `-(number + 1)`. This is the one's complement operation.

For example, `Number.BitwiseNot(0)` returns `-1`, and `Number.BitwiseNot(-1)` returns `0`.

This function is useful in bitmasking scenarios where you need the inverse of a bitmask — for example, clearing all bits that are set in a mask.

## Examples

### Example 1: Bitwise NOT of 0

```powerquery
Number.BitwiseNot(0)
```

### Example 2: Bitwise NOT of a positive integer

### Example 3: Clear bits using NOT as an inverse mask

