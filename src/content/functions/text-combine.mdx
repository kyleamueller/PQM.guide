## Remarks

`Text.Combine` concatenates a list of text values into a single string. Pass a separator to insert a delimiter between each value â€” for example, a comma, space, or newline.

Null values in the list are skipped (not included in the output). If all values are null, the result is an empty string.

## Examples

### Example 1: Combine customer cities

Join all customer cities into a comma-separated string.

Input: `Customers`

```powerquery
let
    Cities = Table.Column(Customers, "City"),
    Combined = Text.Combine(Cities, ", ")
in
    #table({"CityList"}, {{Combined}})
```
