## Remarks

`Number.Exp` computes _e_^x where _e_ is Euler's number (approximately 2.71828). It is the inverse of `Number.Ln`: `Number.Exp(Number.Ln(x)) = x` for any positive x.

Common applications include continuous compounding (finance), exponential growth and decay models, and probability density functions.

`Number.Exp(1)` returns the value of _e_ itself. `Number.Exp(0)` returns `1`.

## Examples

### Example 1: e raised to the power of 1

```powerquery
Number.Exp(1)
```

### Example 3: Continuous compounding — final value of a principal

Compute `P * e^(r*t)` for P=1000, r=0.05, t=3 years.

```powerquery
let
    Principal = 1000,
    Rate = 0.05,
    Time = 3,
    FinalValue = Principal * Number.Exp(Rate * Time)
in
    FinalValue
```
