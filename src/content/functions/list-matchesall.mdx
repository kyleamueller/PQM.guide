## Remarks

`List.MatchesAll` applies a condition function to every item in the list and returns `true` only if every item satisfies the condition. It is the list equivalent of the "for all" quantifier.

Unlike `List.AllTrue`, which expects a pre-computed list of logical values, `List.MatchesAll` accepts any list and a predicate function applied on the fly.

- An empty list returns `true` (vacuous truth).
- The function short-circuits: evaluation stops as soon as a failing item is found.
- Equivalent to `List.AllTrue(List.Transform(list, condition))`, but may short-circuit earlier.

## Examples

### Example 1: Check if all numbers are positive

```powerquery
List.MatchesAll({1, 5, 3, 8}, each _ > 0)
```

### Example 2: Check fails when one item doesn't match

```powerquery
List.MatchesAll({1, 5, -3, 8}, each _ > 0)
```

### Example 3: Validate all order amounts exceed minimum

```powerquery
let
    OrderAmounts = {150.00, 225.50, 89.99, 310.00},
    MinOrder = 50.00,
    AllValid = List.MatchesAll(OrderAmounts, each _ >= MinOrder)
in
    AllValid
```
