## Remarks

`DateTimeZone.ToRecord` decomposes a `datetimezone` value into a record with eight fields: `Year`, `Month`, `Day`, `Hour`, `Minute`, `Second`, `ZoneHours`, and `ZoneMinutes`. This is useful when you need all components at once without calling individual accessor functions. You can expand the resulting record into table columns using `Table.ExpandRecordColumn`.

## Examples

### Example 1: Expand datetimezone components into columns

Decompose EventTimeZone into all its component fields.

Input: `Events`

```powerquery
Table.ExpandRecordColumn(
    Table.AddColumn(
        Table.SelectColumns(Table.FirstN(Events, 3), {"EventID", "EventTimeZone"}),
        "Parts", each DateTimeZone.ToRecord([EventTimeZone])
    ),
    "Parts",
    {"Year", "Month", "Day", "Hour", "Minute", "Second", "ZoneHours", "ZoneMinutes"}
)
```

### Example 2: Decompose a specific datetimezone value

Return the component record for a literal datetimezone.

```powerquery
DateTimeZone.ToRecord(#datetimezone(2024, 3, 15, 14, 30, 0, 5, 30))
```
