## Remarks

`Date.IsInCurrentWeek` returns `true` if the input date falls within the same Sunday-to-Saturday week as today (`DateTime.LocalNow()`). The week boundary is determined using Sunday as the first day of the week (consistent with the default behavior of `Date.StartOfWeek`). The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter this week's orders

Keep only rows where OrderDate falls in the current week.

```powerquery
Table.SelectRows(
    Sales,
    each Date.IsInCurrentWeek([OrderDate])
)
```

### Example 2: Flag current-week rows

Add a boolean column identifying rows from the current week.

```powerquery
Table.AddColumn(
    Sales,
    "IsCurrentWeek", each Date.IsInCurrentWeek([OrderDate]), type logical
)
```
