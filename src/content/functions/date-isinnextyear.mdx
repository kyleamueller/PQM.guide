## Remarks

`Date.IsInNextYear` returns `true` if the input date falls anywhere within the calendar year immediately following the current year (as determined by `DateTime.LocalNow()`). For example, if today is in 2026, this function returns `true` for any date in 2027. The function is re-evaluated on each query refresh.

## Examples

### Example 1: Filter contracts expiring next year

Keep only rows where ExpiryDate is in the next calendar year.

```powerquery
Table.SelectRows(
    Contracts,
    each Date.IsInNextYear([ExpiryDate])
)
```

### Example 2: Flag next-year renewals

Add a boolean column for items expiring next year.

```powerquery
Table.AddColumn(
    Contracts,
    "ExpiresNextYear", each Date.IsInNextYear([ExpiryDate]), type logical
)
```
