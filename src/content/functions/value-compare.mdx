## Remarks

`Value.Compare` returns a signed integer indicating the ordering relationship between two values:

- `-1` — value1 is less than value2
- `0` — the values are equal
- `1` — value1 is greater than value2

This function is useful for building custom sort comparers, range checks, and ordering logic. The return value is compatible with the comparer functions expected by `List.Sort` and related functions.

An optional `comparer` function can be provided for locale-aware or case-insensitive comparisons (e.g., `Comparer.OrdinalIgnoreCase`).

## Examples

### Example 1: Compare numbers

```powerquery
Value.Compare(3, 5)
```

### Example 3: Use as a custom comparer in List.Sort

Sort a list of text values case-insensitively:

```powerquery
List.Sort({"banana", "Apple", "cherry"}, (a, b) => Value.Compare(a, b, Comparer.OrdinalIgnoreCase))
```
