## Remarks

`Number.Log10` returns the common (base-10) logarithm of a number. It is equivalent to `Number.Log(number, 10)`. The result represents the power to which 10 must be raised to produce the given number — for example, `Number.Log10(1000)` returns `3` because 10³ = 1000.

The input must be a positive number. Passing `0` returns negative infinity; a negative number returns `NaN`.

Base-10 logarithms are commonly used for decibel calculations, pH measurements, order-of-magnitude analysis, and log-scale chart axes.

## Examples

### Example 1: Log base 10 of 1000


### Example 3: Determine order of magnitude for revenue values

Input: `Sales`

```powerquery
Table.AddColumn(
    Table.SelectColumns(Table.FirstN(Sales, 4), {"OrderID", "Revenue"}),
    "OrderOfMagnitude", each Number.RoundDown(Number.Log10([Revenue])), type number
)
```
